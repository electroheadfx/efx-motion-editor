       - Compute `deltaFlex = deltaY / pxPerUnit`
       - `seqFlex += deltaFlex`, `propFlex -= deltaFlex`
       - Restore from zero: if `current.seqFlex === 0 && seqFlex > 0 && seqFlex < MIN_RESTORED_FLEX` then `seqFlex = MIN_RESTORED_FLEX` (same for propFlex)
       - Collapse: if `current.seqFlex !== 0 && seqFlex > 0 && seqFlex < COLLAPSE_FLEX_THRESHOLD` then `seqFlex = 0` (same for propFlex)
       - Clamp: `seqFlex = Math.max(0, seqFlex)`, `propFlex = Math.max(0, propFlex)`
       - Return `{ seqFlex, propFlex }`
