<objective>
Fix playhead drag scrubbing and middle-click pan by switching from mouse events to pointer events (UAT Tests 6, 8)

Purpose: Both TimelineInteraction.ts and CanvasArea.tsx call setPointerCapture() but use MouseEvent handlers. MouseEvent has no pointerId property, so the cast `(e as unknown as PointerEvent).pointerId ?? 0` yields 0, and setPointerCapture(0) throws DOMException: InvalidPointerId. Switching to native pointer events provides real pointerId values.

Output: Playhead is draggable for scrubbing. Middle-click pans the preview. Pointer capture works correctly for both interactions.
</objective>
