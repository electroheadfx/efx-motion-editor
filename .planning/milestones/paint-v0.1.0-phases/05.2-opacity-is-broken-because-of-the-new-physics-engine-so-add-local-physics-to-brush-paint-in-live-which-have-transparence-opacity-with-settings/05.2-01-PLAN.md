<objective>
Fix per-stroke opacity layering lost by Phase 05.1 D-12, and unify the render path so all paint goes through wet layer + compositor.

Purpose: Phase 05.1 moved opacity from wet-layer deposit to a global compositor multiplier, which broke per-stroke transparency (all existing wet paint changes brightness when the slider moves). This plan adds a per-pixel strokeOpacity accumulator with Porter-Duff "over" compositing, removes the directRender split, and integrates strokeOpacity into the drying pipeline.

Output: Working per-stroke opacity with correct overlap behavior, unified render path, correct dry baking of transparent strokes.
</objective>
