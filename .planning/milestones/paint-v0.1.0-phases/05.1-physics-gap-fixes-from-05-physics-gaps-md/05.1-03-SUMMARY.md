**Stam stable fluids solver wired into EfxPaintEngine with FluidBuffers allocation, brushTexture removal, compositor opacity piping, and viscosity API**

## Accomplishments
- EfxPaintEngine allocates FluidBuffers (8 Float32Array fields for Stam grid) and FluidConfig with default viscosity/omega_h/darkening
- diffusion.ts physicsStep now delegates directly to fluidPhysicsStep, replacing FBM-displaced darken-flood diffusion
- brushTexture loading removed entirely -- paper-height deposit modulation from Plan 02 replaces brush grain texture
- User opacity flows from brush settings through compositor for per-stroke opacity display
- setViscosity() API exposed for future media type parameterization (watery=0.0001, thick=0.01)
- Visual verification confirmed: organic paint spreading, paper texture influence, subtractive color mixing all working

## Files Created/Modified
- `paint-rebelle-new/src/core/diffusion.ts` - physicsStep now imports and delegates to fluidPhysicsStep from fluids.ts; old diffuseStep marked @deprecated
- `paint-rebelle-new/src/engine/EfxPaintEngine.ts` - FluidBuffers/FluidConfig allocation, loadBrushTexture removed, precomputeDisplacement removed, physicsStep call updated with fluid params, compositeWetLayer receives userOpacity, fluid buffers cleared on reset, setViscosity() API added

## Decisions Made
- Old diffusion functions (diffuseStep, buildColorMap, precomputeDisplacement) kept as @deprecated rather than deleted, for backward compatibility during transition
- FluidConfig defaults: viscosity=0.0001 (watery), omega_h=0.06 (Van Laerhoven), darkening=0.1 (Curtis edge darkening)
- userOpacity derived from `this.state.brushOpts.opacity / 100` at render time

None - plan executed exactly as written.

## Known Trade-offs

**Per-stroke opacity layering at 50% opacity:** Moving opacity to compositor (D-12) means all wet strokes display at the same global opacity rather than per-stroke opacity layering. This is a known trade-off accepted by the user ("it works and its great!"). Per-stroke opacity can be restored in a future phase by storing per-pixel opacity in wet buffers.
