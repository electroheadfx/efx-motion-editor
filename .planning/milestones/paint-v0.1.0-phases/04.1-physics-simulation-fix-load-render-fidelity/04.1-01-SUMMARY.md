## Decisions Made
- Per-channel RGB diffusion with alpha-leading density (D-05 discretion): diffuse wetAlpha using max() flood (higher = more paint), then blend RGB colors weighted by displaced neighbor alpha contributions

- Added `octaves = octaves || 3` default for backward compatibility with existing fbm() callers that pass 2, 3, or 4 octaves
- Kept ensureHeightMap() function (procedural heightmap generation) in codebase since it may be needed as fallback
