**Step 5 — Rewrite replayWithPhysics() for frame-counted diffusion:**

Replace the current `replayWithPhysics()` function. The current version uses timestamp-based fast-forward which is imprecise. The new version uses the recorded `diffusionFrames` count:

```javascript
function replayWithPhysics(strokes) {
  // Pause live physics to prevent non-deterministic interference
  clearInterval(physicsInterval);

  drawBg();
  clearWetLayer();

  // Replay through wet-layer path (NOT directRender) for visual fidelity
  // directRender=false means paint deposits into wet arrays and composites normally

  for (let i = 0; i < strokes.length; i++) {
    const s = strokes[i];

    // Run recorded number of diffusion passes from previous stroke (D-07)
    // This reproduces the organic spreading that happened during live painting
    if (i > 0) {
      const prevDiffFrames = strokes[i - 1].diffusionFrames;
      if (prevDiffFrames && prevDiffFrames > 0) {
        const capped = Math.min(prevDiffFrames, 500); // cap at 50s
        for (let t = 0; t < capped; t++) {
          physicsStep();
        }
      } else if (s.timestamp && strokes[i - 1].timestamp) {
        // Fallback for v1 format: use timestamps (backward compatible)
        const elapsed = s.timestamp - strokes[i - 1].timestamp;
        const ticks = Math.min(Math.floor(elapsed / 100), 500);
        for (let t = 0; t < ticks; t++) {
          physicsStep();
        }
      }
    }

    // Dispatch stroke using same logic as existing replay
    const pts = s.points || s.pts;
    const opts = s.params || s.opts;
    const color = s.color;
    if (s.tool === 'paint') renderPaintStroke(pts, color, opts);
    else if (s.tool === 'liquify') {
      const r = opts.size || opts.radius || 24;
      const w = (opts.waterAmount != null ? opts.waterAmount : (opts.wetness || 35));
      applyLiquifyChunk(pts, r, w / 100 * 0.8 + 0.1);
    }
    else if (s.tool === 'erase') applyEraseStroke(pts, opts);
    else if (s.tool === 'water') applyWaterStroke(pts, opts);
    else if (s.tool === 'smear') applySmearChunk(pts, opts);
    else if (s.tool === 'blend') applyBlendStroke(pts, opts);
    else if (s.tool === 'blow') applyBlowChunk(pts, opts);
    else if (s.tool === 'wet') applyWetChunk(pts, opts);
    else if (s.tool === 'dry') applyDryChunk(pts, opts);
  }

  // Run trailing diffusion from the last stroke (D-07)
  if (strokes.length > 0) {
    const lastDiff = strokes[strokes.length - 1].diffusionFrames;
    if (lastDiff && lastDiff > 0) {
      const capped = Math.min(lastDiff, 500);
      for (let t = 0; t < capped; t++) {
        physicsStep();
      }
    }
  }

  // Final settling pass: a few more ticks for any remaining wet paint
  for (let t = 0; t < 5; t++) physicsStep();

  // Resume live physics
  physicsInterval = setInterval(physicsStep, 100);
}
```

**Per D-08, D-09:** The blow tool should push paint directionally, not just increase wetness. Per the research recommendation, blow adds a temporary per-pixel directional displacement bias that decays each physics tick.

**Step 1 — Add blow velocity arrays (near the diffusion buffer declarations):**
```javascript
// ============================================================
//  BLOW TOOL — directional displacement bias (D-08, D-09)
//  Per-pixel directional force that decays each physics tick.
//  Added to FBM displacement during diffusion for directional push.
// ============================================================
let blowDX = new Float32Array(W * H);
let blowDY = new Float32Array(W * H);
const BLOW_DECAY = 0.92;  // 8% decay per physics tick (~50% after 8 ticks = 0.8s)
const BLOW_STRENGTH = 8.0; // displacement pixels at full strength
```

**Step 2 — Modify applyBlowChunk (currently lines 1321-1359):**

The current blow tool only increases `wetness`. Keep that behavior and ADD directional displacement writing. Inside the inner loop (after the existing `wetness[i] = ...` line), add:

```javascript
        // Directional displacement bias (D-09): push paint in stroke direction
        const blowMask = mask * BLOW_STRENGTH;
        blowDX[i] += ddx * blowMask;
        blowDY[i] += ddy * blowMask;
```

Where `ddx` and `ddy` are the normalized direction vector already computed at the top of the outer loop (`ddx = (p.x - pp.x) / dl`, `ddy = (p.y - pp.y) / dl`). These already exist in the current code.

**Step 3 — Integrate blow displacement into diffuseStep():**

In `diffuseStep()`, modify the displacement lookup to include blow bias. Where the displacement is read:
```javascript
const dx = dispPxX[idx], dy = dispPxY[idx];
```
Change to:
```javascript
const dx = dispPxX[idx] + blowDX[idx];
const dy = dispPxY[idx] + blowDY[idx];
```

This additively combines the static FBM displacement with the temporary blow direction.

**Step 4 — Decay blow displacement in physicsStep():**

In `physicsStep()`, after `dryStep()` and the diffusion frame counter increment, add blow decay:
```javascript
  // Decay blow displacement bias (D-09)
  for (let i = 0; i < W * H; i++) {
    if (blowDX[i] !== 0 || blowDY[i] !== 0) {
      blowDX[i] *= BLOW_DECAY;
      blowDY[i] *= BLOW_DECAY;
      // Zero out very small values to prevent perpetual computation
      if (Math.abs(blowDX[i]) < 0.01) blowDX[i] = 0;
      if (Math.abs(blowDY[i]) < 0.01) blowDY[i] = 0;
    }
  }
```

Note: This loop is O(650K) but the early-continue on zeros makes it fast when no blow is active. For optimization, could track a dirty rect, but not needed for v1.

**Step 5 — Reset blow arrays on clear:**
In `clearWetLayer()`, add:
```javascript
blowDX.fill(0);
blowDY.fill(0);
```
