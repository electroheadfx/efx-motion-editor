    Algorithm (exact from v3):
    1. Get dry canvas ImageData via `dryCtx.getImageData(0, 0, width, height)`
    2. Create display ImageData `displayCtx.createImageData(width, height)`
    3. For each pixel:
       - If `wet.alpha[i] > 1`: compute Beer-Lambert alpha = `255 * (1 - exp(-densityK * wet.alpha[i] / DENSITY_NORM))`, cap at maxDisplayAlpha. Compute wet RGB from wet buffers (wet.r[i], wet.g[i], wet.b[i]). Alpha-blend wet over dry pixel.
       - Else: copy dry pixel directly.
    4. Put display ImageData to displayCtx.

    Import `DENSITY_NORM` from `../types`. Import `clamp` from `../util/math`.

    **render/canvas.ts** ‚Äî Canvas management and drawing from v3:

      - First canvas (dry): `position:relative; z-index:1; cursor:none; touch-action:none; width:100%; height:auto`
