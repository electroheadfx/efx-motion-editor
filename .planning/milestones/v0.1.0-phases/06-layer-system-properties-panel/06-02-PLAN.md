**Class: PreviewRenderer**

```typescript
export class PreviewRenderer {
  private canvas: HTMLCanvasElement;
  private ctx: CanvasRenderingContext2D;
  private imageCache: Map<string, HTMLImageElement>;  // imageId -> loaded HTMLImageElement
  private loadingImages: Set<string>;  // imageIds currently loading
  private videoElements: Map<string, HTMLVideoElement>;  // layerId -> video element

  constructor(canvas: HTMLCanvasElement) { ... }

  /**
   * Render all visible layers for the given frame number.
   * Called on every frame change and on layer property changes.
   * @param fps - frames per second from the active sequence, needed for video layer time sync
   */
  renderFrame(layers: Layer[], frame: number, frameMap: FrameEntry[], fps: number): void { ... }

  /**
   * Get or load an image by imageId from imageStore.
   * Returns HTMLImageElement if cached, null if loading.
   * Uses createImageBitmap pattern for async loading (Pitfall: don't block main thread).
   */
  private getImageSource(imageId: string): HTMLImageElement | null { ... }

  /**
   * Resolve the drawable source for a layer at the given frame.
   * - static-image: always the same HTMLImageElement
   * - image-sequence (base layer): uses frameMap to find imageId for current frame
   * - image-sequence (overlay): per-layer frame indexing via imageIds array
   * - video: HTMLVideoElement with currentTime synced via fps
   */
  private resolveLayerSource(layer: Layer, frame: number, frameMap: FrameEntry[], fps: number): CanvasImageSource | null { ... }

  /**
   * Draw a single layer with its blend mode, opacity, transform, and crop.
   */
  private drawLayer(source: CanvasImageSource, layer: Layer, canvasW: number, canvasH: number): void { ... }

  /** Clean up video elements and caches */
  dispose(): void { ... }
}
```

**renderFrame implementation details:**

1. `ctx.clearRect(0, 0, canvas.width, canvas.height)` — clear canvas
2. Iterate layers bottom-to-top (index 0 first, last index drawn on top)
3. Skip layers where `visible === false`
4. Resolve source via `resolveLayerSource(layer, frame, frameMap, fps)`
5. If source is null (still loading), skip layer (it'll render on next call after load)
6. Call `drawLayer(source, layer, canvas.width, canvas.height)`

**drawLayer implementation details:**

1. `ctx.save()`
2. Map blend mode: `ctx.globalCompositeOperation = blendModeToCompositeOp(layer.blendMode)`
   - `normal` → `source-over`, `screen` → `screen`, `multiply` → `multiply`, `overlay` → `overlay`, `add` → `lighter`
3. `ctx.globalAlpha = layer.opacity`
4. Apply transform: translate to position (center of canvas + layer offset), rotate, scale
   ```
   ctx.translate(layer.transform.x + canvasW / 2, layer.transform.y + canvasH / 2)
   ctx.rotate(layer.transform.rotation * Math.PI / 180)
   ctx.scale(layer.transform.scale, layer.transform.scale)
   ```
5. Draw with crop (9-arg drawImage):
   - Calculate source rectangle from crop fractions (cropTop/Right/Bottom/Left as 0-1 values)
   - Get source dimensions (naturalWidth/naturalHeight for images, videoWidth/videoHeight for video)
   - `ctx.drawImage(source, sx, sy, sw, sh, -dw/2, -dh/2, dw, dh)` — centered around transform origin
   - For layers with no crop (all zeros), use 3-arg `drawImage(source, -dw/2, -dh/2, dw, dh)` for performance
6. `ctx.restore()`

**resolveLayerSource details:**

For `image-sequence` source:
- **Base layer** (`layer.isBase === true` or `layer.source.imageIds.length === 0`): Use `frameMap[frame]` to get `imageId` for the current frame. Call `getImageSource(imageId)` to get HTMLImageElement. If frameMap is empty or frame out of bounds, return null.
- **Overlay image-sequence layer** (`layer.source.imageIds.length > 0`): Per-layer frame indexing. Compute `imageIndex = frame % layer.source.imageIds.length`. Get `imageId = layer.source.imageIds[imageIndex]`. Call `getImageSource(imageId)` to get HTMLImageElement. This ensures overlay image sequences cycle through their imported images frame-by-frame in sync with the playhead (LAYER-02 requirement).

For `static-image` source:
- Call `getImageSource(layer.source.imageId)` directly

For `video` source:
- Get or create a hidden `<video>` element (stored in `videoElements` Map keyed by layer.id)
- Use `assetUrl(layer.source.videoPath)` for the src (via Tauri asset protocol)
- Seek to correct time: `video.currentTime = frame / fps` (fps passed as parameter to renderFrame)
- Return the video element as CanvasImageSource

**getImageSource details:**
- Check `imageCache` Map first — return if found
- If in `loadingImages` Set, return null (load in progress)
- Otherwise: add to `loadingImages`, create new `Image()`, set `src = assetUrl(imageStore.getById(imageId).project_path)`, on `load` event move to `imageCache` and remove from `loadingImages`
- On load completion, trigger a re-render (accept a callback or use a signal)

**Important anti-patterns to avoid (from research):**
- Do NOT recreate video elements every frame — create once, reuse
- Always use `ctx.save()`/`ctx.restore()` around each layer's transform
- Set `globalCompositeOperation` BEFORE `drawImage`, not after
- Translate to center FIRST, then rotate, then scale (Pitfall 6: transform origin)

Implementation:

```typescript
import { useRef, useEffect } from 'preact/hooks';
import { timelineStore } from '../stores/timelineStore';
import { layerStore } from '../stores/layerStore';
import { sequenceStore } from '../stores/sequenceStore';
import { frameMap } from '../lib/frameMap';
import { PreviewRenderer } from '../lib/previewRenderer';
import { playbackEngine } from '../lib/playbackEngine';

export function Preview() {
  const canvasRef = useRef<HTMLCanvasElement>(null);
  const rendererRef = useRef<PreviewRenderer | null>(null);

  // Initialize renderer when canvas mounts
  useEffect(() => {
    const canvas = canvasRef.current;
    if (!canvas) return;

    const renderer = new PreviewRenderer(canvas);
    rendererRef.current = renderer;

    return () => {
      renderer.dispose();
      rendererRef.current = null;
    };
  }, []);

  // Re-render on frame change, layer changes, or frameMap changes
  useEffect(() => {
    const renderer = rendererRef.current;
    if (!renderer) return;

    const frame = timelineStore.currentFrame.value;
    const layers = layerStore.layers.value;
    const frames = frameMap.value;
    const fps = sequenceStore.getActiveSequence()?.fps ?? 24;

    renderer.renderFrame(layers, frame, frames, fps);
  });

  return (
    <div class="relative w-full">
      <div class="relative w-full aspect-video bg-black rounded overflow-hidden">
        <canvas
          ref={canvasRef}
          class="absolute inset-0 w-full h-full"
          style={{ objectFit: 'contain' }}
        />
        {layerStore.layers.value.length === 0 && (
          <div class="absolute inset-0 flex items-center justify-center">
            <span class="text-[var(--color-text-secondary)] text-sm">
              No frames to preview
            </span>
          </div>
        )}
      </div>
    </div>
  );
}
```

**Canvas sizing:** The canvas DOM element is styled to fill its container (via CSS `w-full h-full`). The canvas internal resolution (`canvas.width`, `canvas.height`) should match the display size for preview quality. In the PreviewRenderer, on each renderFrame call, check if the canvas display size has changed and update `canvas.width`/`canvas.height` accordingly:

```typescript
// Inside renderFrame, at the start:
const rect = this.canvas.getBoundingClientRect();
const dpr = window.devicePixelRatio || 1;
const displayW = Math.round(rect.width * dpr);
const displayH = Math.round(rect.height * dpr);
if (this.canvas.width !== displayW || this.canvas.height !== displayH) {
  this.canvas.width = displayW;
  this.canvas.height = displayH;
}
this.ctx.scale(dpr, dpr); // Scale for Retina
// Then composite at rect.width x rect.height logical pixels
```

This follows the existing Retina DPI pattern from TimelineRenderer (Phase 4).
