**Component structure:**

```tsx
export function PropertiesPanel() {
  const selectedId = layerStore.selectedLayerId.value;
  const layers = layerStore.layers.value;
  const selectedLayer = selectedId ? layers.find(l => l.id === selectedId) : null;

  if (!selectedLayer) {
    return (
      <div class="flex items-center justify-center h-14 w-full bg-[#0F0F0F] px-4 shrink-0">
        <span class="text-[10px] text-[var(--color-text-dim)]">
          Select a layer to edit properties
        </span>
      </div>
    );
  }

  return (
    <div class="flex items-center gap-5 h-14 w-full bg-[#0F0F0F] px-4 shrink-0 overflow-x-auto">
      {/* BLEND section */}
      <BlendSection layer={selectedLayer} />
      <div class="w-px h-8 bg-[#2A2A2A]" />
      {/* TRANSFORM section */}
      <TransformSection layer={selectedLayer} />
      {/* CROP section (collapsible or always visible) */}
      <div class="w-px h-8 bg-[#2A2A2A]" />
      <CropSection layer={selectedLayer} />
      {/* FX stub for Phase 7 */}
      {selectedLayer.type === 'video' && !selectedLayer.isBase && (
        <>
          <div class="w-px h-8 bg-[#2A2A2A]" />
          <span class="text-[9px] text-[var(--color-text-dim)] italic">
            FX parameters — Phase 7
          </span>
        </>
      )}
    </div>
  );
}
```

**BlendSection sub-component (inline in same file):**

Contains:
1. **Section label:** `BLEND` in `text-[9px] font-semibold text-[var(--color-text-dimmer)]`

2. **Blend mode dropdown:** `<select>` element with options for all 5 blend modes (normal, screen, multiply, overlay, add). Capitalize labels. Value bound to `selectedLayer.blendMode`. onChange: `layerStore.updateLayer(selectedLayer.id, { blendMode: value as BlendMode })`.

3. **Opacity slider:** `<input type="range" min="0" max="100">` with value `Math.round(selectedLayer.opacity * 100)`. Wire coalescing for undo:
   - `onPointerDown={() => startCoalescing()}`
   - `onPointerUp={() => stopCoalescing()}`
   - `onInput={(e) => { const val = parseInt(value) / 100; layerStore.updateLayer(selectedLayer.id, { opacity: val }); }}`
   - Display percentage: `{Math.round(selectedLayer.opacity * 100)}%`

4. **Visibility toggle:** Button/icon that toggles `visible`. onClick: `layerStore.updateLayer(selectedLayer.id, { visible: !selectedLayer.visible })`. Show filled eye icon for visible, outline eye for hidden.

**TransformSection sub-component (inline in same file):**

Contains:
1. **Section label:** `TRANSFORM`

2. **Position X:** Numeric input. Value: `selectedLayer.transform.x`. onChange updates transform:
   ```typescript
   layerStore.updateLayer(selectedLayer.id, {
     transform: { ...selectedLayer.transform, x: parseFloat(value) || 0 }
   });
   ```
   Use a small `<input type="number">` with step="1". Style with `w-16 text-[11px] bg-[var(--color-bg-input)] text-[#CCCCCC] rounded px-2 py-[5px] outline-none`

3. **Position Y:** Same pattern as X but for `y`.

4. **Scale:** Numeric input. Value: `selectedLayer.transform.scale`. Step: 0.01. Display format: 2 decimal places. onChange updates `transform.scale`.

5. **Rotation:** Numeric input with degree symbol. Value: `selectedLayer.transform.rotation`. Step: 1. onChange updates `transform.rotation`.

**For numeric inputs with drag-style interaction:** Add `onPointerDown`/`onPointerUp` for coalescing. This way dragging the value (if the input supports it) or rapid typing produces a single undo entry.

**CropSection sub-component (inline in same file):**

Contains:
1. **Section label:** `CROP`
2. Four numeric inputs (0-1 range, step 0.01) for cropTop, cropRight, cropBottom, cropLeft
3. Labels: T, R, B, L
4. Same input styling as TransformSection
5. onChange updates the corresponding crop field in transform

**Styling guidelines:**
- Follow the existing dark theme from the current PropertiesPanel mockup
- Input styling: `rounded bg-[var(--color-bg-input)] px-2.5 py-[5px]` with `text-[11px] text-[#CCCCCC]`
- Label styling: `text-[10px] text-[var(--color-text-muted)]`
- Section header: `text-[9px] font-semibold text-[var(--color-text-dimmer)]`
- Separator: `<div class="w-px h-8 bg-[#2A2A2A]" />`
- All controls in a single horizontal row (the panel is h-14 = 56px tall)
