**Change 1 — Larger thumbnails:**

**Change 2 — Hidden scrollbar + wheel-to-horizontal-scroll:**
- On the strip container div (line 69, currently `overflow-x-auto`), add the `scrollbar-hidden` class
- Add an `onWheel` handler to the strip container that converts vertical scroll to horizontal:
  ```typescript
  const handleWheel = useCallback((e: WheelEvent) => {
    if (!stripRef.current) return;
    if (e.deltaY !== 0) {
      e.preventDefault();
      stripRef.current.scrollLeft += e.deltaY;
    }
  }, []);
  ```
- Apply this handler: `onWheel={handleWheel}` on the strip container div

**Change 3 — Replace SortableJS drag with click-to-select + arrow key reorder:**
- Remove the SortableJS import and the entire SortableJS useEffect block (lines 46-65)
- Add state to KeyPhotoStripInner: `const [selectedKpId, setSelectedKpId] = useState<string | null>(null)`
- Make the strip container div focusable: add `tabIndex={0}` and `outline-none` class
- Add a `onKeyDown` handler on the strip container:
  ```typescript
  const handleKeyDown = useCallback((e: KeyboardEvent) => {
    if (!selectedKpId) return;
    const idx = keyPhotos.findIndex(kp => kp.id === selectedKpId);
    if (idx === -1) return;
    if (e.key === 'ArrowLeft' && idx > 0) {
      e.preventDefault();
      sequenceStore.reorderKeyPhotos(sequenceId, idx, idx - 1);
    } else if (e.key === 'ArrowRight' && idx < keyPhotos.length - 1) {
      e.preventDefault();
      sequenceStore.reorderKeyPhotos(sequenceId, idx, idx + 1);
    }
  }, [selectedKpId, keyPhotos, sequenceId]);
  ```
- Pass `selectedKpId` and `onSelect` callback to each KeyPhotoCard
- In KeyPhotoCard: accept `isSelected: boolean` and `onSelect: () => void` props
  - On card click (not on child button clicks), call `onSelect()` to set this card as selected
  - When `isSelected` is true, show a ring border: `ring-2 ring-[var(--color-accent)]`
  - Remove `cursor-grab` from the card class (no longer draggable)
  - Remove the `kp-card` class (was for SortableJS draggable selector)
- When selected card moves via arrow keys, scroll it into view:
  After reorder, use `stripRef.current?.children[newIdx]?.scrollIntoView({ behavior: 'smooth', block: 'nearest', inline: 'nearest' })` (where newIdx is the destination index)
- Clear selection when sequence changes: add effect that resets selectedKpId when sequenceId changes

This approach is better UX for the constrained horizontal space per the user's suggestion in UAT test 5.
