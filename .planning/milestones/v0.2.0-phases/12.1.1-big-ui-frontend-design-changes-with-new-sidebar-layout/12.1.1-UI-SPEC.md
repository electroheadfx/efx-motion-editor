| lg | 24px | Key photo add button width |

## Typography

All text uses Inter font family. Two weights only: 400 (regular) and 600 (semibold). Sizes and colors extracted from the .pen design file.

| Role | Size | Weight | Line Height | Color | Usage |
|------|------|--------|-------------|-------|-------|
| Section title | 11px | 600 (semibold) | 1.2 | `#7B7B94` | SEQUENCES, LAYERS, PROPERTIES headers; letter-spacing: 2px |
| Sequence name | 14px | 600 (semibold) | 1.3 | `#E0E0EE` | Sequence card primary name |
| Sequence meta | 11px | 400 (regular) | 1.2 | `#7B7B94` | "2 keys . 0.6s" metadata line |
| Layer name | 13px | 600 (semibold) | 1.3 | `#E0E0EE` | Layer row primary name (promoted from 500 to 600 for 2-weight consistency) |
| Layer type | 11px | 400 (regular) | 1.2 | `#7B7B94` | "Image", "Sequence . Locked" type labels |
| Field label | 12px | 600 (semibold) | 1.2 | `#7B7B94` | X, Y, Scale, Rot, SX, SY, T, R, B, L, Blur labels (promoted from 500 to 600) |
| Field value | 12px | 400 (regular) | 1.2 | `#E0E0EE` | Numeric input values |
| Blend value | 13px | 600 (semibold) | 1.3 | `#E0E0EE` | Blend mode dropdown display text (promoted from 500 to 600) |
| Button text | 11px | 600 (semibold) | 1.2 | `#9999BB` | "+ Add" button labels (promoted from 500 to 600) |

**Weight consolidation:** Option A applied -- all former weight-500 roles promoted to 600. This creates a clear regular (400) / semibold (600) binary across the sidebar.

**Size scale:** 11px / 12px / 13px / 14px. The 13px size is retained because it is specified in the .pen design for layer names and blend values; removing it would break pixel-perfect fidelity with the canonical design reference.

## Interaction Contracts

### Sub-Window Resizing

| Behavior | Specification |
|----------|---------------|
| Drag start | `pointerdown` on `PanelResizer` ellipsis handle |
| Drag feedback | Panels above and below resize in real-time, no debounce |
| Collapse threshold | Dragging a panel to < 32px height collapses it to 0px (hidden) |
| Restore | Dragging resizer back from collapsed edge restores panel to minimum 80px |
| Persist | Panel height ratios saved to `~/.config/efx-motion/builder-config.yaml` on `pointerup` |
| Cursor | `cursor: grab` on hover, `cursor: grabbing` during drag |
| Independent scroll | Each sub-window has its own overflow-y-auto. Sidebar container itself never scrolls. |

### Sidebar Width Resize

| Behavior | Specification |
|----------|---------------|
| Default width | 317px (from .pen design, stored in `uiStore.sidebarWidth`) |
| Min width | 240px |
| Max width | 480px |
| Drag handle | Invisible full-height zone at sidebar right edge, 8px wide hit area |
| Persist | Width saved to config YAML on `pointerup` |

### Collapse Handle (Double Lines)

| Behavior | Specification |
|----------|---------------|
| Visual | Two vertical 1.5px lines, color `#3c3c51`, 10% of sidebar height, vertically centered |
| Hit zone | Full-height right boundary of sidebar (not just the visible lines) |
| Hover | Subtle brightness increase on the lines (opacity 0.6 -> 1.0 transition, 150ms) |
| Click | Toggles `uiStore.sidebarCollapsed` signal |
| Collapsed state | Double lines render at left window boundary; click to expand |
| Replaces | Current `<<` chevron button is removed entirely |

### Key Photos Inline

| Behavior | Specification |
|----------|---------------|
| Placement | Key photo thumbnails appear directly under the selected sequence inside its card |
| Auto-collapse | Selecting a different sequence auto-collapses the previous one's key photos |
| Auto-expand | Newly selected sequence auto-expands its key photos |
| Scrub auto-select | Releasing timeline scrub on a different sequence triggers auto-select |
| Animation | Height transition ~150ms, ease-out curve |
| + button | 24px wide, centered Lucide `plus` icon, opens full Imported media view (not filtered); `aria-label="Add key photo"` |
| Thumbnail size | 56px height, aspect-ratio preserved width |
| Frame badge | Bottom-right overlay badge showing hold frames (e.g. "11f", "4f") |
| Empty slot | Placeholder with `#3A3A50` background |

### Keyframe Navigation Bar

| Behavior | Specification |
|----------|---------------|
| Layout | Horizontal: `< + trash >` with 4px gaps |
| < button | Navigate playhead to previous keyframe. Disabled (grayed `#7B7B94` at 40% opacity) when at first keyframe or no keyframes. |
| > button | Navigate playhead to next keyframe. Disabled when at last keyframe or no keyframes. |
| + button | Add keyframe when NOT on a keyframe. Show update icon (Lucide `refresh-cw` or `pencil`) when ON a keyframe. |
| Trash button | Lucide `trash-2` icon. Grayed out unless playhead is exactly on a keyframe. Deletes current keyframe. |
| Hover | Light background color change on hover for each button (`--sidebar-input-bg` at 80% opacity) |
| Blur field | Same row as keyframe nav bar, right-aligned, 16px gap between nav bar and blur field |

### Interpolation Mode Swap

| Behavior | Specification |
|----------|---------------|
| Trigger | A keyframe diamond is selected on the timeline (`keyframeStore.selectedKeyframeFrames` has entries) |
| Effect | Blend mode dropdown + opacity slider row is replaced by inline interpolation controls |
| Controls | Clickable chips/buttons for: Linear, Ease In, Ease Out, Ease In-Out, Cubic |
| Active indicator | Selected curve has `--sidebar-selected-layer-bg` background |
| Deselect diamond | Blend+opacity row returns |
| Old popover | Double-click-to-show-interpolation-menu behavior is removed entirely |

### Custom Scrollbars

| Property | Value |
|----------|-------|
| Width | 4px |
| Track | Transparent (invisible) |
| Thumb | `#4A4A64`, rounded |
| Auto-hide | Thumb visible only while scrolling (CSS `:hover` on parent or `scrollbar-gutter: stable` + opacity transition) |
