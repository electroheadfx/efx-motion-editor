<decisions>
## Implementation Decisions

### Icon Selection
- **Playback controls (timeline + canvas):** Play, Pause, SkipBack (step back), SkipForward (step forward), Rewind (seek to start)
- **Zoom controls (timeline + canvas):** Plus (zoom in), Minus (zoom out)
- **Canvas specific:** Maximize (fullscreen), Maximize2/Minimize2 (fit-lock toggle — Maximize2 when unlocked, Minimize2 when locked, accent color still shows lock state)
- **Toolbar:** FilePlus (New), FolderOpen (Open), Save (Save), CircleSlash or Ban (Blur Off bypass toggle — orange when active), Images (Imported), Settings (Settings), Download (Export)
- **Timeline 'Fit All':** Becomes an icon (Claude picks appropriate shrink/fit icon)

### Icon Sizing (varies by area)
- Timeline panel: 14-16px (compact bar)
- Canvas bottom bar: 16-18px (more spacious)
- Toolbar: 16px (icon-only buttons)

### Visual Style
- Stroke width: default (2) — standard lucide weight
- Color: inherit text color via currentColor (automatic theme support through CSS variables)
- Canvas play/pause button keeps its emphasis treatment: larger, round (rounded-full), accent color — primary action stands out
- All other buttons uniform within their area

### Text vs Icon-Only
- **All buttons become icon-only** across all three areas (timeline, canvas, toolbar)
- Text displays (timecode, zoom level %, zoom multiplier x) remain as text — they're read-only displays, not buttons
- Toolbar buttons (New, Open, Save, Blur Off, Imported, Settings, Export) all become icon-only

### Tooltips
- All icon-only buttons get hover tooltips
- Tooltip format: "Label (Shortcut)" — e.g., "Play (Space)", "Zoom In (=)", "Save (⌘S)"
- Helps users learn keyboard shortcuts through natural discovery

### Claude's Discretion
- Exact icon for timeline "Fit All" replacement
- Fine-tuning of icon sizes within the ranges specified per area
- Hover state transitions (opacity, bg color changes)
- Tooltip delay timing and positioning
- Unsaved indicator placement relative to new Save icon

</decisions>
