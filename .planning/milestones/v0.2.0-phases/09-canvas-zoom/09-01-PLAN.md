  <name>Task 1: Create canvasStore.ts with zoom/pan signals and preset logic</name>
  <files>Application/src/stores/canvasStore.ts</files>
  <action>
Create `Application/src/stores/canvasStore.ts` following the exact pattern of `timelineStore.ts` (module-scoped Preact signals exported via an object literal with methods).

**Signals:**
- `zoom = signal(1)` -- current zoom level (0.1 to 4.0)
- `panX = signal(0)` -- horizontal pan offset in px
- `panY = signal(0)` -- vertical pan offset in px
- `containerWidth = signal(0)` -- measured container width (set by ResizeObserver in CanvasArea)
- `containerHeight = signal(0)` -- measured container height
- `zoomPercent = computed(() => Math.round(zoom.value * 100))` -- reactive percent for display

**Constants:**
- `MIN_ZOOM = 0.1`
- `MAX_ZOOM = 4.0`
- `ZOOM_PRESETS = [0.1, 0.25, 0.5, 0.75, 1.0, 1.5, 2.0, 3.0, 4.0]` -- per user decision, 9 stops

**Methods:**

`zoomIn()` -- Find next preset ABOVE current zoom (with epsilon 0.001 for float comparison). If current zoom is between presets (e.g., 1.37 from wheel), snap to next preset up (1.5). If at max, no-op. Zoom is center-anchored: scale panX/panY proportionally (`pan * newZoom/oldZoom`).

`zoomOut()` -- Find next preset BELOW current zoom (reverse search with epsilon). If between presets (e.g., 1.37), snap to next preset down (1.0). If at min, no-op. Center-anchored same as zoomIn.

`setSmoothZoom(newZoom: number, cursorX: number, cursorY: number)` -- For wheel/pinch (smooth, no snapping). Clamp newZoom to [MIN_ZOOM, MAX_ZOOM]. Adjust pan to keep point under cursor stable: `pan = pan * (newZoom/oldZoom) + cursor * (1 - newZoom/oldZoom) / newZoom`. Update zoom signal.

`setPan(x: number, y: number)` -- Direct pan setter.

`fitToWindow()` -- Calculate fit zoom using containerWidth/containerHeight vs the canvas natural size. The canvas element has `max-w-[830px]` with `aspect-video` (16:9). At zoom=1, the canvas renders at its natural CSS size (up to 830px wide). Fit zoom = scale where the canvas fits the container:
  - `canvasNaturalW = Math.min(containerWidth.peek(), 830)`
  - `canvasNaturalH = canvasNaturalW / (projectStore.width.peek() / projectStore.height.peek())`
  - If `canvasNaturalH > containerHeight.peek()`, recalculate: `canvasNaturalH = containerHeight.peek()`, `canvasNaturalW = canvasNaturalH * aspectRatio`
  - `fitScale = Math.min(containerWidth.peek() / canvasNaturalW, containerHeight.peek() / canvasNaturalH, 1.0)`
  - But actually simpler: at zoom=1 the canvas fills naturally. If the container is large enough, fit=1. If too small, fit < 1. The key insight: just set zoom to 1.0 and let CSS handle it... BUT with zoom, we need the CSS transform to scale. So fit should be `min(availW / canvasNaturalW, availH / canvasNaturalH)` clamped to max 1.0.
  - Actually, re-examining: the canvas at zoom=1 already has `max-w-[830px]` constraint. The transform: scale() MULTIPLIES this. So at zoom=0.5, it's half size. At zoom=2, it's double (may overflow). Fit = zoom where the scaled canvas matches the container. Since at zoom=1 the canvas already uses max-w-[830px] aspect-video which auto-sizes to container, fit is simply 1.0 UNLESS the container is too small for the max-w-[830px] element. In practice: `fitScale = Math.min(containerW / canvasRenderedW, containerH / canvasRenderedH)` where canvasRenderedW/H is the ACTUAL rendered size of the inner div at zoom=1.
  - SIMPLEST correct approach: store the actual rendered canvas dimensions (from the inner div's getBoundingClientRect at zoom=1 equivalent). For now, use: `fitScale = Math.min(containerW / Math.min(containerW, 830), containerH / (Math.min(containerW, 830) / aspectRatio))`. Since `min(containerW, 830)` is the natural width, `containerW / min(containerW, 830)` is always >= 1 when containerW >= 830. So clamp to max 1.0.
  - Final implementation: `const natW = Math.min(containerW, 830); const natH = natW / aspectRatio; fitScale = Math.min(containerW / natW, containerH / natH, MAX_ZOOM); zoom.value = Math.max(MIN_ZOOM, Math.min(fitScale, 1.0));`
  - Import `projectStore` for `width`/`height` signals.
  - Reset panX=0, panY=0 on fit (per user decision: "Pan resets to centered on Fit").

`updateContainerSize(w: number, h: number)` -- Called by ResizeObserver in CanvasArea. Sets containerWidth/containerHeight. Does NOT auto-recalculate fit (user may be zoomed manually).

`reset()` -- Reset all signals to defaults: zoom=1, panX=0, panY=0.

`isAtMinZoom = computed(() => zoom.value <= ZOOM_PRESETS[0] + 0.001)` -- For disabling - button
`isAtMaxZoom = computed(() => zoom.value >= ZOOM_PRESETS[ZOOM_PRESETS.length - 1] - 0.001)` -- For disabling + button

Export as `export const canvasStore = { ... }`.

3. **Refactor handleWheel** to delegate to canvasStore:
   ```
   const handleWheel = useCallback((e: WheelEvent) => {
     if (!(e.metaKey || e.ctrlKey)) return;
     e.preventDefault();
     const container = e.currentTarget as HTMLElement;
     const rect = container.getBoundingClientRect();
     const cursorX = e.clientX - rect.left - rect.width / 2;
     const cursorY = e.clientY - rect.top - rect.height / 2;
     const oldZoom = canvasStore.zoom.peek();
     const factor = e.deltaY > 0 ? 0.9 : 1.1;
     const newZoom = oldZoom * factor;
     canvasStore.setSmoothZoom(newZoom, cursorX, cursorY);
   }, []);
   ```

4. **Refactor handlePointerDown/Move/Up** to use `canvasStore.panX`/`panY` instead of local signals. Same logic, just different signal source.

5. **Replace handleFit** with `canvasStore.fitToWindow()`.

6. **Add ResizeObserver** using `useEffect` on the outer container div (the `overflow-hidden` flex container). Use a `useRef` for the container. In the effect:

   - On mount, also call `canvasStore.fitToWindow()` to set initial fit.
   - Cleanup: disconnect observer on unmount.
   - Add `ref={containerRef}` to the outer scrollable div.

     - Calculate `newZoom = canvasStore.zoom.peek() * ge.scale`.
     - Call `canvasStore.setSmoothZoom(newZoom, cursorX, cursorY)`.
   - Attach listeners with `{passive: false}` to allow preventDefault.
   - Cleanup: remove both listeners on unmount.
