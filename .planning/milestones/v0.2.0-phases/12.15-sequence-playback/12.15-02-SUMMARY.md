## Accomplishments
- Timeline renderer draws orange (#E5841B) stroke border around full frame range of isolated sequences
- Name labels show solid orange background with white text for isolated and hovered sequences
- Click on name label toggles isolation, sets active sequence, clears FX selection
- Hover on name label shows pointer cursor and triggers orange highlight preview
- Shared getNameLabelRect geometry helper eliminates code duplication between renderer and interaction

## Decisions Made
- Isolation border renders BEFORE the name overlay block so the label draws on top of the border edge -- provides visual layering where the label is always readable
- Name label hover state (hoveredNameLabelSequenceId) is managed internally by the renderer via setHoveredNameLabel(), same pattern as setHoveredKeyframe -- avoids adding hover to the effect subscription which would cause unnecessary redraws from external signal changes
