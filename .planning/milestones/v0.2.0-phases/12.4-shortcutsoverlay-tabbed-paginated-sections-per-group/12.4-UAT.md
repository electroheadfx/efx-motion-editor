## Tests

### 1. Sections tab is default on open
expected: Press `?` to open the ShortcutsOverlay. It opens on the Sections tab (first tab, index 0) showing 7 clickable rows with group names and shortcut counts.
result: pass

### 2. Click group row to jump to tab
expected: On the Sections tab, click any group row (e.g. "Canvas"). The overlay switches to that group's tab showing only its shortcut entries (Canvas should show 8 entries).
result: pass

### 3. Group tab shows correct entries
expected: Click the "Playback" tab in the tab bar. It shows 5 shortcut entries (Play/Pause, Step frame, J, K, L). Click "Keyframes" tab — shows 1 entry (Add keyframe at current frame).
result: pass

### 4. Arrow Left/Right cycle tabs with wrapping
expected: From any tab, press Arrow Right to advance to the next tab, Arrow Left to go back. Wrapping works: Right from Keyframes (tab 7) goes to Sections (tab 0), Left from Sections goes to Keyframes (tab 7).
result: pass

### 5. Tab key advances tabs
expected: Press the Tab key — it advances to the next tab, behaving identically to Arrow Right, including wrapping.
result: pass

### 6. Number keys 1-7 jump to group tabs
expected: From any tab, press number keys 1 through 7. Each jumps directly to the corresponding group: 1=Playback, 2=File, 3=Editing, 4=Navigation, 5=Canvas, 6=Blur, 7=Keyframes.
result: pass

### 7. Arrow Up/Down navigate Sections rows
expected: On the Sections tab, press Arrow Down to move the highlight down through the 7 rows. It wraps from Keyframes back to Playback. Arrow Up moves highlight upward with wrapping. The focused row has a visible background highlight.
result: pass

### 8. Enter jumps from Sections to focused group
expected: On the Sections tab, use Arrow Down to highlight a group (e.g. "Navigation"), then press Enter. The overlay switches to that group's tab.
result: pass

### 9. Modal height stability across tabs
expected: Switch between tabs with different entry counts (e.g. Canvas with 8 entries vs Keyframes with 1 entry). The modal does not visibly jump in height — the content area has a stable minimum height.
result: pass

### 10. Reset to Sections on reopen
expected: Navigate to any non-Sections tab, close the overlay (Escape or backdrop click), then reopen with `?`. The overlay starts on the Sections tab (index 0), not the previously viewed tab.
result: pass

### 11. Theme compatibility
expected: Cycle themes with Shift+Cmd+T (close overlay first). Reopen overlay — active tab accent border, text colors, kbd badges, and Sections row highlights render correctly in all themes. No hardcoded colors visible.
result: pass
