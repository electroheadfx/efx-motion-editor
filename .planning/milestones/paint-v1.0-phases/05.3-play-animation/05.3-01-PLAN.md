**Private fields:**
- `config: AnimationConfig | null = null`
- `playing = false`
- `currentFrame = 0`
- `rafId = 0`
- `lastFrameTime = 0`
- `frameStrokes: FrameStroke[] = []` (precomputed frame distribution)

**Public methods:**

1. `play(config: AnimationConfig): void` (per D-03, D-05, D-11)
   - Store config, set playing = true, currentFrame = 0
   - Call `this.engine.setInputLocked(true)` (per D-11: painting disabled during playback)
   - Call `this.engine.setAnimationMode(true)` (pause engine render loop)
   - Get strokes via `this.engine.getStrokes()`
   - Call `this.distributeStrokes(strokes, config.frameCount)` to precompute frame mapping
   - Set `this.lastFrameTime = 0`
   - Start loop: `this.rafId = requestAnimationFrame(this.tick)`

2. `stop(): void` (per D-04: stop shows final painting)
   - Set `this.playing = false`
   - `cancelAnimationFrame(this.rafId)`
   - Call `this.engine.renderAllStrokes()` to render complete painting (per D-04)
   - Call `this.engine.setAnimationMode(false)` (resume engine render loop)
   - Call `this.engine.setInputLocked(false)` (unlock painting)

3. `isPlaying(): boolean` returns `this.playing`

4. `getState(): AnimationState` returns `{ playing: this.playing, currentFrame: this.currentFrame, totalFrames: this.config?.frameCount ?? 0 }`

**Private methods:**

5. `distributeStrokes(strokes: PaintStroke[], totalFrames: number): void` (per D-01)
   - If no strokes, set `this.frameStrokes = []` and return
   - Find `minTime` = first stroke timestamp, `maxTime` = last stroke timestamp
   - `duration = maxTime - minTime || 1` (avoid division by zero for single stroke)
   - For each stroke:
     - `ratio = (stroke.timestamp - minTime) / duration`
     - `startFrame = Math.floor(ratio * (totalFrames - 1))`
     - `pointCount = stroke.points.length`
     - Per D-02 (point-by-point): calculate frames needed = `Math.max(1, Math.ceil(pointCount / 8))` (baseline ~8 points per frame)
     - `endFrame = Math.min(totalFrames - 1, startFrame + framesNeeded - 1)`
     - `pointsPerFrame = Math.ceil(pointCount / (endFrame - startFrame + 1))`
     - Push `{ stroke, startFrame, endFrame, pointsPerFrame }` to `this.frameStrokes`

6. `private tick = (timestamp: number): void =>` (arrow function for correct `this`)
   - If `!this.playing || !this.config` return
   - If `this.lastFrameTime === 0` set `this.lastFrameTime = timestamp`
   - `frameDuration = 1000 / this.config.fps`
   - If `timestamp - this.lastFrameTime >= frameDuration`:
     - `this.lastFrameTime = timestamp - ((timestamp - this.lastFrameTime) % frameDuration)` (drift correction)
     - Call `this.renderFrame(this.currentFrame)`
     - Call `this.config.onFrame?.(this.currentFrame, this.engine.getDisplayCanvas())` (per D-06)
     - Increment `this.currentFrame`
     - If `this.currentFrame >= this.config.frameCount`:
       - Set `this.playing = false`
       - Call `this.engine.renderAllStrokes()` (ensure final frame shows complete painting)
       - Call `this.engine.setAnimationMode(false)`
       - Call `this.engine.setInputLocked(false)`
       - Call `this.config.onComplete?.()`
       - Return (per D-05: play once, stop at last frame)
   - `this.rafId = requestAnimationFrame(this.tick)`

7. `private renderFrame(frameIndex: number): void`
   - This method renders all strokes up to and including those active at `frameIndex`.
   - Call `this.engine.renderAllStrokes()` first to clear and prepare canvas — BUT we need a variant that renders only UP TO a frame.
   - **Implementation approach:** Instead of renderAllStrokes (which renders everything), implement frame-limited rendering:
     - Call a new engine method `renderStrokesUpTo(strokes, pointLimit?)` OR
     - Build the strokes-to-render list here and pass to a targeted engine method.
   - **Simpler approach that works with existing architecture:** Collect all strokes that should be fully rendered (endFrame < frameIndex) and all strokes partially rendered at frameIndex (startFrame <= frameIndex <= endFrame). For each partial stroke, compute how many points to include based on frame position within that stroke's range.
   - Build a `strokesToRender: Array<{ stroke: PaintStroke; pointCount: number }>` list:
     - For each FrameStroke where `endFrame < frameIndex`: include with full point count
     - For each FrameStroke where `startFrame <= frameIndex && endFrame >= frameIndex`: include with `pointsPerFrame * (frameIndex - startFrame + 1)` points (capped at total)
     - Skip strokes where `startFrame > frameIndex`
   - Call `this.engine.renderPartialStrokes(strokesToRender)` (new engine method, see Task 2)

```typescript
/** Render strokes up to specified point counts — used by AnimationPlayer for progressive frame rendering (per D-02) */
renderPartialStrokes(strokeData: Array<{ stroke: PaintStroke; pointCount: number }>): void {
  // Clear canvas to background
  this.bgData = drawBg(this.bgCtx, this.state.bgMode, this.width, this.height, this.paperTextures, this.userPhoto)
  this.dualCanvas.dryCtx.drawImage(this.bgCanvas, 0, 0)
  clearWetLayer(this.wet, this.savedWet, this.drying.dryPos, this.blowDX, this.blowDY, this.lastStrokeMask)
  this.fluid.u.fill(0); this.fluid.v.fill(0)
  this.fluid.u0.fill(0); this.fluid.v0.fill(0)
  this.fluid.p.fill(0); this.fluid.div.fill(0)

  const sampleHFn = (x: number, y: number) => sampleH(this.paperHeight, x, y, this.width, this.height)

  for (const { stroke: a, pointCount } of strokeData) {
    // Slice points to the requested count (progressive rendering per D-02)
    const pts = pointCount >= a.points.length ? a.points : a.points.slice(0, pointCount)
    const opts = a.params
    const color = a.color

    if (a.tool === 'paint' && color) {
      renderPaintStroke(
        pts, color, opts,
        this.dualCanvas.dryCtx, this.wet, this.savedWet,
        this.drying.dryPos, this.lastStrokeMask,
        this.paperHeight, this.brushGrain,
        this.width, this.height,
        this.state.hasPenInput, this.state.wetPaper,
        this.state.embossStrength, this.state.embossStack,
        opts.waterAmount / 100,
        sampleHFn,
      )
    } else if (a.tool === 'erase') {
      applyEraseStroke(
        pts, opts,
        this.dualCanvas.dryCtx, this.wet,
        this.width, this.height,
        this.state.hasPenInput,
        this.state.embossStrength,
        this.paperHeight,
        this.state.bgMode,
        this.bgData,
      )
    }
    // Force dry after each stroke (same pattern as redrawAll — no physics ticks)
    forceDryAll(this.wet, this.savedWet, this.drying, this.dualCanvas.dryCtx, this.width, this.height)
  }

  // Composite wet layer for display (so getDisplayCanvas shows current frame)
  const displayCtx = this.dualCanvas.displayCtx
  displayCtx.clearRect(0, 0, this.width, this.height)
  compositeWetLayer(displayCtx, this.wet, this.width, this.height, sampleHFn)
}
```

This method follows the exact same pattern as `redrawAll()` but takes explicit stroke data with point count limits. It imports are already available (renderPaintStroke, applyEraseStroke, forceDryAll, etc. are already imported at the top of EfxPaintEngine.ts).
