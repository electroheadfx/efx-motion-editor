**Helper function `getElementLabel` (per D-03):**
```typescript
function getElementLabel(el: PaintElement, index: number): string {
  const toolLabels: Record<string, string> = {
    brush: 'Brush',
    eraser: 'Eraser',
    line: 'Line',
    rect: 'Rectangle',
    ellipse: 'Ellipse',
    fill: 'Fill',
  };
  return `${toolLabels[el.tool] || el.tool} ${index + 1}`;
}
```

**Component `StrokeList`:**

1. Read reactive state:
   - `const frame = timelineStore.currentFrame.value;`
   - `const _pv = paintStore.paintVersion.value;` (subscribe to re-renders)
   - `const paintFrame = paintStore.getFrame(layerId, frame);`
   - `const elements = paintFrame?.elements ?? [];`
   - `const selectedIds = paintStore.selectedStrokeIds.value;`

2. Display order: **reversed** (last element at top = front-most, matching LayerList convention and standard graphics app behavior per Research Pitfall 6). Create `const displayElements = [...elements].reverse();` and `const totalElements = elements.length;`.

3. CollapsibleSection wrapper (per D-01, D-02):
   - `const strokesCollapsed = useSignal(false);`
   - Title: `` `STROKES (${totalElements})` ``
   - Wrap the list content in `<CollapsibleSection title={...} collapsed={strokesCollapsed}>`

4. SortableJS integration (per D-12, matching LayerList pattern):
   - `const listRef = useRef<HTMLDivElement>(null);`
   - `useEffect` creates `Sortable.create(listRef.current, { animation: 150, ghostClass: 'opacity-30', handle: '.stroke-drag-handle', forceFallback: true, fallbackClass: 'opacity-30', onEnd(evt) { ... } })` and returns cleanup `() => instance.destroy()`.
   - In `onEnd`: revert DOM mutation (LayerList pattern), convert visual indices to array indices: `const fromIdx = totalElements - 1 - oldIndex; const toIdx = totalElements - 1 - newIndex;`, then call `paintStore.reorderElements(layerId, frame, fromIdx, toIdx)` followed by `paintStore.refreshFrameFx(layerId, frame)`.
   - Dependency array: `[totalElements, layerId, frame]`

5. Selection anchor ref for Shift+click (per D-09, Research Pitfall 5):
   - `const selectionAnchor = useRef<number>(-1);`
   - Updated on plain click and Cmd+click.

6. Selection handler `handleSelect(elementId: string, displayIndex: number, e: MouseEvent)` (per D-09):
   - **Plain click**: `paintStore.clearSelection(); paintStore.selectStroke(elementId); selectionAnchor.current = displayIndex;`
   - **Cmd+click** (`e.metaKey`): Toggle -- if `selectedIds.has(elementId)` then `paintStore.deselectStroke(elementId)` else `paintStore.selectStroke(elementId)`. Update anchor: `selectionAnchor.current = displayIndex;`
   - **Shift+click** (`e.shiftKey`): Range select from `selectionAnchor.current` to `displayIndex`. `paintStore.clearSelection();` then for each index in range (inclusive), `paintStore.selectStroke(displayElements[i].id);`

7. Visibility toggle handler `handleToggleVisibility(elementId: string)` (per D-07, D-08):
   - Read current: `const el = elements.find(e => e.id === elementId);`
   - Toggle: `paintStore.setElementVisibility(layerId, frame, elementId, el?.visible === false);`
   - After toggle, refresh FX: `paintStore.refreshFrameFx(layerId, frame);`

8. Delete handler `handleDelete(elementId: string)` (per STRK-03):
   - `paintStore.removeElement(layerId, frame, elementId);`
   - `paintStore.deselectStroke(elementId);`
   - `paintStore.refreshFrameFx(layerId, frame);`

9. Auto-scroll on canvas selection change (per D-11):
   - `useEffect` that watches `selectedIds`. If size > 0, find first selected ID, query `listRef.current?.querySelector([data-element-id="${firstId}"])`, call `row?.scrollIntoView({ block: 'nearest', behavior: 'smooth' })`.

10. Row rendering -- for each `displayElements[i]`:
    - Outer div: `class="group/row flex items-center gap-1.5 rounded px-2 py-1 cursor-pointer select-none"`, `data-element-id={el.id}`, backgroundColor: selected ? `var(--sidebar-selected-layer-bg)` : `transparent`, opacity: hidden ? 0.4 : 1.
    - Drag handle: `<div class="stroke-drag-handle cursor-grab shrink-0 opacity-60 hover:opacity-100"><GripVertical size={12} style={{color: 'var(--sidebar-resizer-icon)'}} /></div>`
    - Eye toggle: `<button class="w-4 h-4 flex items-center justify-center shrink-0 rounded hover:bg-[#ffffff10]" onClick={(e) => { e.stopPropagation(); handleToggleVisibility(el.id); }}>` renders `EyeOff` if hidden (size 12, secondary color, opacity 0.4) or `Eye` if visible (size 12, secondary color).
    - Color swatch (per D-04): `<div class="w-2.5 h-2.5 rounded-full shrink-0" style={{ backgroundColor: el.color }} />`
    - Label (per D-03): `<span class="flex-1 truncate text-[11px]" style={{ color: isSelected ? 'var(--sidebar-text-primary)' : 'var(--sidebar-collapse-line)' }}>{getElementLabel(el, totalElements - 1 - i)}</span>` -- uses `totalElements - 1 - i` because `getElementLabel` adds 1 internally (`index + 1`), so this produces correct 1-based labels: front-most element (i=0 in reversed display) gets label N, back-most gets label 1.
    - Delete button: `<button class="w-4 h-4 flex items-center justify-center rounded shrink-0 opacity-0 group-hover/row:opacity-100" style={{color: 'var(--sidebar-text-secondary)'}} onClick={(e) => { e.stopPropagation(); handleDelete(el.id); }}><X size={12} /></button>`
    - Click handler on the row: `onClick={(e) => handleSelect(el.id, i, e as MouseEvent)}`

11. Empty state: If `totalElements === 0`, render `<div class="px-3 py-2 text-[10px]" style={{color: 'var(--sidebar-text-secondary)'}}>No strokes on this frame</div>` inside the CollapsibleSection.

12. List container: `<div ref={listRef} class="flex flex-col gap-0.5 max-h-48 overflow-y-auto">` for scrollable list with max height.

Export the component as named export: `export function StrokeList({ layerId }: StrokeListProps) { ... }`
