  <name>Task 1: Create LayerList component with SortableJS drag-and-drop, visibility toggle, and delete</name>
  <files>Application/src/components/layer/LayerList.tsx</files>
  <action>
Create `Application/src/components/layer/LayerList.tsx`. This component renders the layer stack for the active sequence with:

1. **Layer items displayed top-to-bottom** (highest layer first in the visual list = last in the array). Reverse the array for display: `[...layerStore.layers.value].reverse()` so that the topmost layer visually is at the top of the list, and the base layer is at the bottom.

2. **SortableJS integration** following the EXACT pattern from `SequenceList.tsx`:
   - `useRef` for the container element
   - `useEffect` to create/destroy Sortable instance
   - `animation: 150`
   - `handle: '.layer-drag-handle'` — drag handle element on each layer row
   - `filter: '.layer-base'` — prevent dragging the base layer
   - `onMove(evt)` callback: return `false` if `evt.related.classList.contains('layer-base')` to prevent dropping anything below the base layer
   - `onEnd(evt)` callback: convert visual indices (reversed) back to array indices and call `layerStore.reorder(fromIndex, toIndex)`. Since the list is displayed reversed, the mapping is: `arrayIndex = totalLayers - 1 - visualIndex`. So: `const from = totalLayers - 1 - evt.oldIndex; const to = totalLayers - 1 - evt.newIndex;` then `layerStore.reorder(from, to)`.

3. **Each layer row renders:**
   - Drag handle (`layer-drag-handle` class) — a small grip icon. Hidden for base layer.
   - Visibility toggle button — eye icon that toggles `layer.visible`. On click: `layerStore.updateLayer(layer.id, { visible: !layer.visible })`. Use unicode symbols or simple circles (filled=visible, outline=hidden).
   - Color-coded type indicator (small square): `image-sequence` = blue, `static-image` = teal, `video` = purple
   - Layer name (truncated)
   - Layer type + blend mode text (small secondary text)
   - Delete button (X icon) — only shown for non-base layers. On click: `layerStore.remove(layer.id)`.
   - Row click selects the layer: `layerStore.setSelected(layer.id)` + `uiStore.selectLayer(layer.id)`

4. **Selected layer highlight:** If `layerStore.selectedLayerId.value === layer.id`, apply a highlight background color (e.g., `bg-[#2A2A3A]` or `border-l-2 border-[var(--color-accent)]`).

5. **Base layer styling:** The base layer row has the `layer-base` CSS class (for SortableJS filter), no drag handle, no delete button, and a subtle "locked" indicator.

**Styling:** Follow the existing dark theme patterns from LeftPanel.tsx. Use the same text sizes (`text-[11px]`, `text-[9px]`), colors (`var(--color-text-muted)`, `#AAAAAA`, `#CCCCCC`), and backgrounds (`#252525`, `#1E1E1E`).
