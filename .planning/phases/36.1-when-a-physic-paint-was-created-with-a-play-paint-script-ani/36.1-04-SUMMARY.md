## Decisions Made

- Local Play scrubbing updates standalone preview state and optional cached frame display only; it does not call the editor frame sync path.
- The workflow strip still exposes the existing inspection callback, but Play lane click handling routes through `onPreviewPlayFrame` and avoids `onNavigateToSyncedFrame`.
- Save play uses the selected saved script start frame (`playStartFrame`) as the payload start and rendered-frame app-frame base, preventing local in-between preview position from shifting the saved range.
