    **Part D: Create Project I/O Service**

    Create `Application/src-tauri/src/services/project_io.rs` with:

    1. `create_project_dir(dir_path: &str)` -- creates the project directory with images/ and images/.thumbs/ subdirectories. Returns Ok(()) or error.

    2. `save_project(project: &MceProject, file_path: &str, project_root: &str)` -- converts absolute image paths to relative, serializes to JSON, writes atomically (write to .tmp, rename). The `project_root` is the directory containing the .mce file (derived from file_path's parent).

    3. `open_project(file_path: &str)` -- reads .mce file, deserializes, converts relative image paths back to absolute using the file's parent directory as project root. Returns MceProject with absolute paths for frontend use.

    4. `make_relative(abs_path: &str, project_root: &str) -> String` -- strips project_root prefix from absolute path, returns relative.

    5. `make_absolute(rel_path: &str, project_root: &str) -> String` -- joins project_root with relative path, returns absolute.

    **IMPORTANT path handling:** The .mce file stores RELATIVE paths (`relative_path`, `thumbnail_relative_path` fields on MceImageRef). When saving, convert from absolute to relative. When opening, convert from relative to absolute. The `project_root` is ALWAYS the parent directory of the .mce file path.

    **Atomic write pattern:** Write to `{file_path}.tmp`, then `fs::rename` to `{file_path}`. This prevents corruption if the app crashes mid-write.

    ```rust
    use tauri::command;
    use crate::models::project::{ProjectData, MceProject, MceSequence, MceImageRef};
    use crate::services::project_io;

    /// Legacy command -- kept for backward compatibility
    #[command]
    pub fn project_get_default() -> ProjectData {
        ProjectData {
            name: "Untitled Project".into(),
            fps: 24,
            width: 1920,
            height: 1080,
        }
    }

    /// Create a new project: makes directory structure, returns initial MceProject
    #[command]
    pub fn project_create(name: String, fps: u32, dir_path: String) -> Result<MceProject, String> {
        // Create project directory structure
        project_io::create_project_dir(&dir_path)?;

        let now = chrono::Utc::now().to_rfc3339();
        Ok(MceProject {
            version: 1,
            name,
            fps,
            width: 1920,
            height: 1080,
            created_at: now.clone(),
            modified_at: now,
            sequences: vec![],
            images: vec![],
        })
    }

    /// Save project to .mce file (atomic write)
    #[command]
    pub fn project_save(project: MceProject, file_path: String) -> Result<(), String> {
        let project_root = std::path::Path::new(&file_path)
            .parent()
            .ok_or_else(|| "Invalid file path: no parent directory".to_string())?
            .to_str()
            .ok_or_else(|| "Invalid file path: non-UTF8 characters".to_string())?;

        project_io::save_project(&project, &file_path, project_root)
    }

    /// Open project from .mce file
    #[command]
    pub fn project_open(file_path: String) -> Result<MceProject, String> {
        project_io::open_project(&file_path)
    }
    ```
