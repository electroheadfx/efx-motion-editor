## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Real-key cache -> generated cache | Trusted real-key alpha PNGs are read to create derived render-only frames. |
| Store mutation -> renderer/UI subscribers | Cache mutations must notify visual subscribers without exposing generated frames as editable state. |
| Canvas/data URL decode -> real-key save path | Browser image/canvas failures must not roll back real-key mutations. |
