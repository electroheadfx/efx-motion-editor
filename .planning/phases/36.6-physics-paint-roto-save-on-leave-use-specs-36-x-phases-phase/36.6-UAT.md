### 2. Leave Dirty Frame Saves Automatically

### 3. Rapid Roto Navigation Opens Latest Destination

result: [pending]

### 5. Manual Save Current Remains Isolated
expected: Press Save current directly on a dirty Roto frame. It should keep its normal label when idle, become disabled only while that save is running, and should not navigate away after the manual save completes.

expected: If save-on-leave fails because the paint window cannot apply the result or times out, the original source frame should remain marked dirty, the previous cached output should remain intact, and the app should not open the queued destination as if the save succeeded.
