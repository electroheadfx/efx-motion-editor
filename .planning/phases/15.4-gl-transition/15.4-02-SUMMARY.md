## Decisions Made
- Offscreen canvases for dual-capture are reused between frames via module-level cache with size-check (avoids allocation per frame)
- GL transition rendering path is checked before cross-dissolve, with break on match to avoid double rendering
- TransitionProperties returns null for gl-transition selection since Plan 03 handles the GL transition properties panel
- Transition preview images are programmatically generated gradients (warm orange-red vs cool blue) rather than bundled assets
- Added `applyDisabled` prop to ShaderDetail to disable Apply button when no next sequence exists (D-03)
