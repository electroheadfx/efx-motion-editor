    **1. Add `computeCrossDissolveOpacity` to `Application/src/lib/transitionEngine.ts`:**

    ```typescript
    /**
     * Compute the cross dissolve opacity pair for a given global frame.
     * Returns [outgoingOpacity, incomingOpacity] where both sum to ~1.0.
     * outgoingOpacity: opacity of the sequence that's fading out (1.0 -> 0.0)
     * incomingOpacity: opacity of the sequence that's fading in (0.0 -> 1.0)
     */
    export function computeCrossDissolveOpacity(
      globalFrame: number,
      overlapStart: number,     // global frame where overlap begins
      overlapDuration: number,  // number of overlap frames
      curve: EasingType,
    ): [outgoingOpacity: number, incomingOpacity: number] {
      if (overlapDuration <= 0) return [1, 0];
      const framesIntoOverlap = globalFrame - overlapStart;
      const t = Math.max(0, Math.min(1, framesIntoOverlap / overlapDuration));
      const eased = applyEasing(t, curve);
      return [1 - eased, eased];
    }
    ```

    **2. Add `crossDissolveOverlaps` computed signal to `Application/src/lib/frameMap.ts`:**

    This computed signal derives overlap zones from content sequences that have `crossDissolve` set. It does NOT modify the existing `frameMap` or `trackLayouts` computations for now — it's a parallel signal.

    ```typescript
    export interface CrossDissolveOverlap {
      outgoingSequenceId: string;  // sequence that's fading out
      incomingSequenceId: string;  // sequence that's fading in
      overlapStart: number;        // global frame where overlap begins
      overlapEnd: number;          // global frame where overlap ends (exclusive)
      duration: number;            // frames of overlap
      curve: EasingType;           // from the crossDissolve transition
    }

    export const crossDissolveOverlaps = computed<CrossDissolveOverlap[]>(() => {
      const seqs = sequenceStore.sequences.value.filter(s => s.kind === 'content');
      const overlaps: CrossDissolveOverlap[] = [];

      // Build sequential layout of content sequences
      let currentFrame = 0;
      const seqRanges: { seq: Sequence; start: number; end: number }[] = [];
      for (const seq of seqs) {
        const totalFrames = seq.keyPhotos.reduce((sum, kp) => sum + kp.holdFrames, 0);
        if (totalFrames === 0) continue;
        seqRanges.push({ seq, start: currentFrame, end: currentFrame + totalFrames });
        currentFrame += totalFrames;
      }

      // Find cross dissolves between adjacent sequences
      for (let i = 0; i < seqRanges.length - 1; i++) {
        const outgoing = seqRanges[i];
        const incoming = seqRanges[i + 1];
        if (outgoing.seq.crossDissolve) {
          const cd = outgoing.seq.crossDissolve;
          const halfDuration = Math.floor(cd.duration / 2);
          // Cross dissolve centered on boundary (per D-15)
          const boundary = outgoing.end;
          const overlapStart = boundary - halfDuration;
          const overlapEnd = boundary + (cd.duration - halfDuration);
          overlaps.push({
            outgoingSequenceId: outgoing.seq.id,
            incomingSequenceId: incoming.seq.id,
            overlapStart,
            overlapEnd,
            duration: cd.duration,
            curve: cd.curve,
          });
        }
      }

      return overlaps;
    });
    ```

    Import `EasingType` from `../types/layer` and `Sequence` from `../types/sequence` if needed.

        if (otherLocalFrame < 0 || otherLocalFrame >= otherSeqFrames.length) continue;

        // Apply keyframe interpolation to other sequence's layers
        const otherLayers = otherSeq.layers.map(layer => {
          if (!layer.keyframes || layer.keyframes.length === 0) return layer;
          if (isFxLayer(layer) || layer.isBase) return layer;
          const values = interpolateAt(layer.keyframes, otherLocalFrame);
          if (!values) return layer;
          return {
            ...layer,
            opacity: values.opacity,
            transform: { ...layer.transform, x: values.x, y: values.y, scaleX: values.scaleX, scaleY: values.scaleY, rotation: values.rotation },
            blur: values.blur,
          };
        });

        // Render the other sequence with its cross-dissolve opacity
        renderer.renderFrame(otherLayers, otherLocalFrame, otherSeqFrames, otherSeq.fps, false, otherOpacity);

        break; // Only one cross dissolve can be active at a time
      }

    **IMPORTANT:** The above is a simplified approach. The primary render already happened at line 68 (or the fade-modified version). For cross dissolve, we need BOTH sequences rendered with their respective opacities. The cleaner approach:

    When a cross dissolve is active at the current frame:
    1. Clear the canvas
    2. Render outgoing sequence with `outOpacity` (using `sequenceOpacity` parameter)
    3. Render incoming sequence ON TOP with `inOpacity` (using `clearCanvas=false, sequenceOpacity=inOpacity`)

    This means the primary render logic needs to be aware of cross dissolve and modify its behavior. The executor should refactor the primary render to:
    - Check if in cross dissolve zone BEFORE the primary render
    - If yes: render outgoing at outOpacity, then incoming at inOpacity, skip the normal render
    - If no: proceed with normal fade-aware render

    This is complex — the executor should read the full Preview.tsx to find the right insertion pattern.
