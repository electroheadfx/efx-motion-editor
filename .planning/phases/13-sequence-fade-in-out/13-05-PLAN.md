    What was built: Complete transition system with fade in, fade out, and cross dissolve.

    How to verify:
    1. Open the editor with a project that has 2+ content sequences with key photos
    2. FADE IN TEST:
       - Select the first sequence in the sidebar
       - In the TRANSITIONS section, click "Add Fade In"
       - Verify: timeline shows gradient overlay at sequence start (dark left, clear right) with white border and diagonal line
       - Verify: label "In" appears centered in the overlay
       - Scrub through the fade zone — preview should show content fading from transparent/invisible to fully visible
       - In sidebar, change Duration to 24 frames — overlay width should update
       - Change Mode to "Solid Color" — Color picker appears
       - Set Color to red (#FF0000) — preview should show red-to-content fade
       - Change Curve to "Linear" — fade should be linear (not eased)
       - Click "Remove Transition" — overlay disappears
       - Cmd+Z — transition restored
    3. FADE OUT TEST:
       - Click "Add Fade Out" on same sequence
       - Verify: overlay at sequence end (clear left, dark right), diagonal top-left to bottom-right
       - Scrub through — content fades out to transparent/black
       - Verify: sequence name shifted right past fade-in overlay (if both present)
    4. CROSS DISSOLVE TEST:
       - On first sequence, click "Add Cross Dissolve" (should only appear when next sequence exists)
       - Verify: overlay centered on pink boundary between sequences
       - Verify: two-zone gradient (left darker, right lighter)
       - Verify: pink boundary marker visible through overlay transparency
       - Scrub through dissolve zone — Seq 1 fades out while Seq 2 fades in simultaneously
       - In sidebar, change Duration to 48 — overlay width doubles
    5. SELECTION TEST:
       - Click fade in overlay on timeline — border becomes full white, sidebar shows TransitionProperties
       - Click empty space — deselects
       - Click fade out — selects that one
       - Press Delete key — fade out removed
    6. FX LAYER TEST:
       - If FX layers exist, verify fade in/out overlays render on FX bars too (smaller, per D-03)
    7. PERSISTENCE TEST:
       - Save project (Cmd+S)
       - Close and reopen — all transitions preserved
    8. SHORT SEQUENCE TEST:
       - On a very short sequence, verify transition label hides when too narrow
       - Verify sequence name clips or hides per D-21
