## Decisions Made

- **Generated interpolation metadata remains disposable cache:** load/replacement code can use persisted generated metadata as cache evidence, but source truth is still real source keys plus normalized settings/overrides.
- **Inference is scoped to hydration/replacement:** normal settings changes do not invent new overrides; this prevents legacy compact/global segments from unexpectedly becoming custom-timed.
- **Explicit overrides are source-endpoint durable:** settings changes preserve stored `fromSourceFrame`/`toSourceFrame` overrides unless the caller explicitly replaces them.
