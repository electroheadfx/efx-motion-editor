    **Part A: Add VelocityCache to motionBlurEngine.ts**

    Add to the existing `motionBlurEngine.ts` (after the existing functions from Plan 01):

    ```typescript
    /** Per-layer velocity cache for delta computation (per D-13) */
    export class VelocityCache {
      private prev = new Map<string, KeyframeValues>();
      private lastFrame = -1;

      /**
       * Compute velocity for a layer. Returns null if no previous frame cached
       * (first frame or after seek — skip blur for that frame per Pitfall 2).
       */
      computeForLayer(
        layerId: string,
        currentValues: KeyframeValues,
        currentFrame: number,
      ): LayerVelocity | null {
        // Invalidate cache on non-sequential frame (seek detection per Pitfall 6)
        if (Math.abs(currentFrame - this.lastFrame) > 1) {
          this.prev.clear();
        }
        this.lastFrame = currentFrame;

        const previous = this.prev.get(layerId);
        // Store current for next frame
        this.prev.set(layerId, { ...currentValues });

        if (!previous) return null;
        return computeLayerVelocity(currentValues, previous);
      }

      /** Clear all cached values (call on project load, sequence change) */
      clear() {
        this.prev.clear();
        this.lastFrame = -1;
      }
    }
    ```

    3. In the `renderFrame()` method, find the per-layer rendering loop. For each content layer (not FX, not paint, not generator, not adjustment — check via `isFxLayer`, `isGeneratorLayer`, `isAdjustmentLayer`, and `layer.type !== 'paint'`), AFTER the existing blur pass block and the normal `drawLayer` call, add a motion blur pass:

       The motion blur pass should be added as an alternative rendering path. The logic is:
       - If motion blur is enabled AND layer has keyframes AND layer is a content layer (not fx/generator/adjustment/paint):
         a. Compute current KeyframeValues via `interpolateAt(layer.keyframes, localFrame)` where `localFrame` is the sequence-local frame
         b. Call `this.velocityCache.computeForLayer(layer.id, values, globalFrame)` to get velocity
         c. If velocity is null or `isStationary(velocity)`, skip blur (render normally per D-05/MBLR-08)
         d. Otherwise: render to offscreen canvas (reuse `getBlurOffscreen`), apply motion blur via `applyMotionBlur(off.canvas, ctx, {dx: velocity.dx, dy: velocity.dy}, motionBlurStore.getStrength(), motionBlurStore.getSamples(), logicalW, logicalH)`, composite the result
       - If applyMotionBlur returns false (WebGL2 unavailable), fall through to normal rendering

       **Integration point (per D-12):** The motion blur pass goes AFTER layer content is rendered to the offscreen canvas and BEFORE compositing to the main canvas. This is the same point where the Gaussian blur pass currently lives (lines 346-363). The two effects can coexist: if a layer has both blur AND is moving, apply Gaussian blur first, then motion blur.

       Concrete insertion: Within the content layer rendering block, after the existing `if (blurRadius > 0 && !blurStore.isBypassed())` branch, add an `else if` for motion blur, plus a case where both apply. The structure should be:

       ```typescript
       const wantMotionBlur = motionBlurStore.isEnabled()
         && motionBlurStore.getSamples() > 0
         && layer.keyframes && layer.keyframes.length > 0
         && !isFxLayer(layer) && !isGeneratorLayer(layer) && !isAdjustmentLayer(layer)
         && layer.type !== 'paint';

       let motionVelocity: {dx: number; dy: number} | null = null;
       if (wantMotionBlur) {
         const kfValues = interpolateAt(layer.keyframes!, localFrame);
         if (kfValues) {
           const vel = this.velocityCache.computeForLayer(layer.id, kfValues, frame);
           if (vel && !isStationary(vel)) {
             motionVelocity = { dx: vel.dx, dy: vel.dy };
           }
         }
       }

       if (blurRadius > 0 && !blurStore.isBypassed()) {
         // existing gaussian blur path...
         // After applyBlurToCanvas, add motion blur if needed:
         if (motionVelocity) {
           applyMotionBlur(off.canvas, off.ctx, motionVelocity,
             motionBlurStore.getStrength(), motionBlurStore.getSamples(),
             off.canvas.width, off.canvas.height);
         }
         // then composite as before
       } else if (motionVelocity) {
         // Motion blur only (no gaussian blur)
         const off = this.getBlurOffscreen(Math.round(logicalW), Math.round(logicalH));
         if (off) {
           off.ctx.clearRect(0, 0, off.canvas.width, off.canvas.height);
           off.ctx.save();
           this.drawLayerToOffscreen(source, layer, off.ctx, logicalW, logicalH);
           off.ctx.restore();
           applyMotionBlur(off.canvas, off.ctx, motionVelocity,
             motionBlurStore.getStrength(), motionBlurStore.getSamples(),
             off.canvas.width, off.canvas.height);
           ctx.save();
           ctx.globalCompositeOperation = blendModeToCompositeOp(layer.blendMode);
           ctx.globalAlpha = effectiveOpacity;
           ctx.drawImage(off.canvas, 0, 0, logicalW, logicalH);
           ctx.restore();
         }
       } else {
         // Normal rendering (existing drawLayer call)
         this.drawLayer(source, layer, logicalW, logicalH, sequenceOpacity);
       }
       ```
