## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Standalone window -> editor Tauri event bus | Rendered output apply payloads and apply-result feedback cross between webviews. |
| Project file -> hydrated editor state | Serialized layer data becomes runtime editor layer/source identity. |
| User save path -> filesystem | User-selected path receives editable physics paint state JSON. |
| Browser fallback -> user downloads | Non-Tauri Save state uses browser primitives and reports outcome in UI. |
