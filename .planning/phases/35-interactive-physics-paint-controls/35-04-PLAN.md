## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Standalone apply event -> editor mutation | External window payload mutates app frame store. |
| Editor apply result -> standalone UI | App validation result crosses back to external window for user feedback. |
| Rendered frame data -> preview canvas | Data URLs/image refs become drawable canvas sources. |
