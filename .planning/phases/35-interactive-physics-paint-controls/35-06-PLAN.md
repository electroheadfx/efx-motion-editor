## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Standalone window -> editor Tauri event bus | Rendered output apply payload and apply-result feedback cross between windows. |
| Project file -> hydrated editor state | Serialized layer data becomes trusted editor runtime layer state after load. |
| User save path -> filesystem | User-chosen path receives editable physics paint JSON from the standalone toolbar. |
| Browser fallback -> user downloads | Non-Tauri Save state uses browser download primitives and must expose success/error state. |
