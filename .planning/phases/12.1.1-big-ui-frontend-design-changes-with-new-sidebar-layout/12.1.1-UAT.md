## Tests

### 1. Sidebar 3-Panel Layout
expected: The sidebar shows three stacked panels — SEQUENCES, LAYERS, PROPERTIES — each with a collapsible header and ChevronDown icon.
result: pass

### 2. Panel Resize via Drag
expected: Dragging the horizontal resize handles between panels resizes them vertically. Dragging a panel below ~32px collapses it to zero height (content hidden). Dragging a collapsed panel's handle back up restores it.
result: pass

### 3. Sidebar Width Resize
expected: Dragging the right edge of the sidebar resizes its width. Width is clamped between 240px and 480px.
result: pass

### 4. Sidebar Collapse Handle
expected: A double-line collapse handle is visible at the right edge of the sidebar. Clicking it collapses the entire sidebar. Clicking again restores it.
result: pass

### 5. Sequence Cards
expected: Sequences display as bordered cards with rounded corners and a left accent bar. Selected sequence has a distinct border color vs unselected.
result: pass

### 6. Inline Key Photos
expected: Expanding a sequence card reveals inline key photo thumbnails with a smooth slide animation (~150ms). An "+" button opens the key photo importer.
result: pass

### 7. Layer Row Icons
expected: Layer rows show a GripVertical drag handle on the left, an Eye icon for visibility toggle (Eye/EyeOff on click), a Lock icon on the base layer, and an X delete button. Rows are ~44px tall.
result: pass

### 8. Keyframe Navigation Bar
expected: Select a keyframe diamond on the timeline. The sidebar properties panel shows a navigation bar with < (prev), + (add), trash (delete), > (next) buttons. Clicking prev/next jumps the playhead to adjacent keyframes. Buttons disable at boundaries (no prev on first, no next on last).
result: pass

### 9. Inline Interpolation Chips
expected: With a keyframe diamond selected, the properties panel swaps blend+opacity controls for 4 easing preset chips (interpolation types). Clicking a chip selects that easing. The old double-click popover on timeline diamonds should no longer appear.
result: pass

### 10. Layout Persistence
expected: Resize the sidebar width and panel heights. Close and reopen the app. The sidebar width and panel heights are restored to the values you set.
result: pass
