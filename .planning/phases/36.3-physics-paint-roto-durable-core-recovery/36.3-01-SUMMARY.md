## Decisions Made

- Cached Roto output is the durable truth; reopen uses a reference image rather than restoring editable stroke state.
- Clean cached references must not be sent into the physics engine as background content because that can make saved pixels look like unsaved editable paint.
- Navigation away and back preserves the saved cached reference when no new unsaved edits were made.
- Existing saved outputs can synthesize `real-key` Roto metadata on load so the durable core remains usable for projects saved before the metadata fix.
- The unreliable close path remains deferred and is not treated as part of 36.3-01 completion; the validated core path is Save current plus project quit/reload.
