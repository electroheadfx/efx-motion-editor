Create `Application/src/lib/bezierPath.ts` as a pure math module (no store imports, no side effects). Import BezierAnchor from types/paint.ts, import fitCurve from 'fit-curve', import { Bezier } from 'bezier-js'. Export all functions below.

**1. cubicBezierPoint(x0, y0, x1, y1, x2, y2, x3, y3, t) -> {x, y}**
Standard cubic bezier evaluation using de Casteljau formula. See RESEARCH.md Pattern 2 code example.

**2. pointsToBezierAnchors(points: [number, number, number][], tolerance = 4.0) -> BezierAnchor[]**
Per D-04. Convert freehand [x,y,pressure] arrays to BezierAnchor[]. Use fitCurve for the core fitting. For pressure, use nearest-point lookup from original points array (findNearestPressure helper, private). Clamp result: if anchors > 40, retry with tolerance * 1.5 up to 3 times. If anchors < 3 and points > 10, retry with tolerance * 0.5. See RESEARCH.md Pattern 1 code for implementation reference.

**3. shapeToAnchors(shape: PaintShape) -> { anchors: BezierAnchor[], closedPath: boolean }**
Per D-03, D-06. Handle three shape types:
- `line`: 2 anchors, null handles, closedPath=false
- `rect`: 4 corner anchors, null handles (straight segments), closedPath=true. Use min/max of x1/x2, y1/y2. Apply rotation if shape.rotation is set (rotate corner positions around shape center).
- `ellipse`: 4 quadrant anchors with kappa=0.5522847498 handles, closedPath=true. Apply rotation if present.
All anchors get pressure=0.5. Import PaintShape type from types/paint.ts.

**4. sampleBezierPath(anchors: BezierAnchor[], spacing = 2.0, closedPath = false) -> [number, number, number][]**
Re-densify bezier anchors back to [x,y,pressure] points for perfect-freehand rendering. For each pair of adjacent anchors, compute cubic bezier with handleOut of A and handleIn of B (null handle = anchor position). Use Math.max(10, Math.ceil(distance / spacing)) steps. Interpolate pressure linearly between anchors. If closedPath, also sample from last anchor back to first anchor. See RESEARCH.md Pattern 2 sampleBezierPath.

**5. insertAnchorOnSegment(anchors: BezierAnchor[], segmentIndex: number, t: number) -> BezierAnchor[]**
Per D-11. Use `new Bezier(...)` from bezier-js, call `.split(t)`. Create new anchor at split point with interpolated pressure. Update handleOut of anchor A and handleIn of anchor B. Return new array with the inserted anchor. See RESEARCH.md Pattern 4 code.

**6. deleteAnchor(anchors: BezierAnchor[], idx: number) -> BezierAnchor[]**
Per D-12. Do not delete if length <= 2. Splice out the anchor. Auto-adjust neighboring handles: point handles toward each other at 1/3 distance for smooth reconnection. See RESEARCH.md Anchor Deletion code.

**7. updateCoupledHandle(anchor: BezierAnchor, draggedSide: 'in' | 'out', newPos: {x, y}, isAltHeld: boolean) -> void**
Per D-08. Set the dragged handle to newPos. If isAltHeld, set cornerMode=true and return (break tangent). If cornerMode is already true, return (stays broken). Otherwise mirror opposite handle: compute angle from anchor to dragged handle, set opposite at angle+PI, preserve opposite handle's original distance from anchor. See RESEARCH.md Smooth Handle Coupling code.

**8. dragSegment(anchorA: BezierAnchor, anchorB: BezierAnchor, t: number, targetPos: {x, y}) -> void**
Per D-10. Compute current point at t on the bezier between A and B. Get delta to targetPos. Distribute delta to handleOut of A (weight 1-t) and handleIn of B (weight t). See RESEARCH.md Segment Dragging code.

**9. findNearestSegment(anchors: BezierAnchor[], point: {x, y}, closedPath = false) -> { segmentIndex: number, t: number, distance: number } | null**
For each pair of adjacent anchors (and last->first if closedPath), create a Bezier instance and call `.project({x, y})` to get nearest point. Return the segment with smallest distance, its index, and the t parameter. Return null if anchors.length < 2.

**10. hitTestAnchor(anchors: BezierAnchor[], point: {x, y}, radius: number) -> { anchorIndex: number, part: 'anchor' | 'handleIn' | 'handleOut' } | null**
Check each anchor: distance to anchor point, then to handleIn (if not null), then to handleOut (if not null). Return first match within radius, with 'anchor' taking priority over handles. Return null if nothing hit.
