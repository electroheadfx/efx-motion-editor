# Phase 12.13: Linear-timeline - Context

**Gathered:** 2026-03-19
**Status:** Ready for planning
**Source:** Direct user input

<domain>
## Phase Boundary

This phase adds a "linear timeline" layout mode where all sequences are displayed on a single horizontal row instead of the current vertically-stacked track layout. The user can toggle between the existing stacked view and this new linear/flat view.

</domain>

<decisions>
## Implementation Decisions

### Timeline Layout Mode Toggle
- Add a toggle that switches between "stacked" (current default, sequences on separate rows) and "linear" (all sequences on the same row)
- In linear mode, sequences appear side-by-side horizontally on a single track row
- Sequence names appear as overlay text on top of thumbnails in linear mode

### Thumbnail Display Options
- Add an option to control what is shown per sequence block: "thumb+name" (default) or "thumb only"
- "thumb+name": shows the thumbnail with the sequence name overlaid
- "thumb only": shows just the thumbnail without the name overlay

### Claude's Discretion
- How to render the toggle UI (button, dropdown, or icon toggle in the timeline panel toolbar)
- How to store the layout mode preference (uiStore or timelineStore)
- Visual styling of the name overlay (font size, background, opacity, position)
- How linear mode interacts with FX/content-overlay sequences (render all flat, or only content sequences)
- Scroll behavior in linear mode (horizontal scroll only, no vertical scroll needed)
- Interaction with zoom — how zooming works when all sequences share one row

</decisions>
