**5. Throttle slider onChange with requestAnimationFrame:**

Add a `useRef` for the rAF id at the top of the component:
```typescript
const rafSlider = useRef(0);
```

In `renderSlider`, change the range input's onInput handler:

```typescript
onInput={(e) => {
  const v = parseFloat((e.target as HTMLInputElement).value);
  if (rafSlider.current) cancelAnimationFrame(rafSlider.current);
  rafSlider.current = requestAnimationFrame(() => {
    onInput(v);
    rafSlider.current = 0;
  });
}}
```

This prevents multiple state updates per frame during fast dragging.
