**2. Calculate clamped position:**
Inside the component, calculate the modal position. Assume modal is approximately 300px wide and 400px tall (measure actual rendered size if possible, or use refs):
```typescript
const MODAL_WIDTH = 300;
const MODAL_HEIGHT = 420;
const MARGIN = 12; // keep 12px from window edges

const left = mouseX != null
  ? Math.min(Math.max(mouseX, MARGIN), window.innerWidth - MODAL_WIDTH - MARGIN)
  : (window.innerWidth - MODAL_WIDTH) / 2;  // fallback: center

const top = mouseY != null
  ? Math.min(Math.max(mouseY, MARGIN), window.innerHeight - MODAL_HEIGHT - MARGIN)
  : (window.innerHeight - MODAL_HEIGHT) / 2;  // fallback: center
```

  // ... other existing styles
}}>
