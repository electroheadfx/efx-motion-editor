```typescript
let _gl: WebGL2RenderingContext | null = null;
let _glCanvas: HTMLCanvasElement | null = null;
let _initFailed = false;

function getGL(): WebGL2RenderingContext | null {
  if (_initFailed) return null;
  if (_gl) return _gl;
  _glCanvas = document.createElement('canvas');
  const gl = _glCanvas.getContext('webgl2', {
    premultipliedAlpha: false,
    preserveDrawingBuffer: true,
    antialias: false,
    depth: false,
    stencil: false,
  });
  if (!gl) { _initFailed = true; return null; }
  _glCanvas.addEventListener('webglcontextlost', (e) => {
    e.preventDefault();
    _gl = null;
  });
  _gl = gl;
  return _gl;
}
```

**getGL() function:** Follow the exact glBlur.ts pattern:
- Create offscreen canvas via `document.createElement('canvas')`
- Get webgl2 context with `{ premultipliedAlpha: false, preserveDrawingBuffer: true, antialias: false, depth: false, stencil: false }`
- Listen for webglcontextlost, on loss: set `_gl = null`, clear all cached programs and FBOs
- Set `_initFailed = true` if context creation fails

**compileShader(gl, type, src):** Compile a shader, log error on failure, return null on failure.

**linkProgram(gl, vertSrc, fragSrc):** Compile vert+frag, link program, validate, return null on failure.

**ensureResources(gl, w, h):** Called at start of render. If _currentW !== w or _currentH !== h, resize the glCanvas and recreate all framebuffer textures at the new dimensions. Create programs on first call:
- _stampProgram from STAMP_VERT_SRC + STAMP_FRAG_SRC
- _compositeProgram from BRUSH_FX_VERT_SRC + buildSpectralCompositeSrc()
- _grainProgram from BRUSH_FX_VERT_SRC + buildGrainPostSrc()
- _edgeDarkenProgram from BRUSH_FX_VERT_SRC + EDGE_DARKEN_POST_FRAG_SRC
- _bleedProgram from BRUSH_FX_VERT_SRC + buildBleedPostSrc()

Create FBO textures: For each FBO (stroke, accumA, accumB, post), create a RGBA8 texture at w x h, attach as COLOR_ATTACHMENT0. Check framebuffer completeness.

**createTexture(gl, w, h):** Helper to create a GL_TEXTURE_2D with RGBA8 format, LINEAR filtering, CLAMP_TO_EDGE wrapping.

**createFBO(gl, texture):** Helper to create framebuffer with the given texture as color attachment.

**setupQuadVAO(gl):** Create VAO with position buffer [-1,-1, 1,-1, -1,1, 1,1] as triangle strip. This is the fullscreen quad for post-passes.

**setupStampVAO(gl):** Create VAO with position buffer [-1,-1, 1,-1, -1,1, 1,1] and texcoord buffer [0,0, 1,0, 0,1, 1,1] for stamp quad rendering.

**disposeBrushFx():** Export. Deletes all GL resources, sets all cached refs to null, removes glCanvas.

**hexToGLColor(hex, opacity):** Convert hex string "#RRGGBB" to [r, g, b, a] float array in linear RGB (not sRGB). Use formula: linearR = pow(sRGBr, 2.2). This is important because spectral mixing operates in linear space.

Add the main rendering function to brushFxRenderer.ts:

**renderStyledStrokes(strokes: PaintStroke[], width: number, height: number): HTMLCanvasElement | null**

Export this function. It renders an array of styled (non-flat) PaintStroke objects to the offscreen WebGL2 canvas and returns the canvas element (or null if GL unavailable).

The caller (paintRenderer.ts, in a future plan) will do `ctx.drawImage(result, 0, 0)` to composite.

**Algorithm:**

```typescript
export function renderStyledStrokes(
  strokes: PaintStroke[],
  width: number,
  height: number,
): HTMLCanvasElement | null {
  const gl = getGL();
  if (!gl || !_glCanvas) return null;
  ensureResources(gl, width, height);
  if (!_stampProgram || !_compositeProgram) return null;

  // Clear accumulation buffer A
  gl.bindFramebuffer(gl.FRAMEBUFFER, _accumFBO_A);
  gl.clearColor(0, 0, 0, 0);
  gl.clear(gl.COLOR_BUFFER_BIT);

  let readAccum = _accumFBO_A;
  let readAccumTex = _accumTex_A;
  let writeAccum = _accumFBO_B;
  let writeAccumTex = _accumTex_B;

  for (const stroke of strokes) {
    if (stroke.tool === 'eraser') continue; // Erasers handled by Canvas 2D path

    // 1. Render stroke stamps to _strokeFBO
    renderStrokeStamps(gl, stroke, width, height);

    // 2. Composite _strokeTex onto accumulation buffer using spectral mix
    compositeSpectral(gl, readAccumTex!, _strokeTex!, writeAccum!, width, height);

    // 3. Swap ping-pong buffers
    [readAccum, writeAccum] = [writeAccum, readAccum];
    [readAccumTex, writeAccumTex] = [writeAccumTex, readAccumTex];
  }

  // 4. Apply post-effects based on stroke params
  // Determine which post-effects are needed from stroke params
  const lastStrokeParams = aggregatePostEffectParams(strokes);
  let currentTex = readAccumTex!;
  let currentFBO = readAccum;

  if (lastStrokeParams.edgeDarken > 0.01) {
    applyPostEffect(gl, _edgeDarkenProgram!, currentTex, _postFBO!, width, height, { u_edgeDarken: lastStrokeParams.edgeDarken });
    currentTex = _postTex!;
    // For further passes, swap textures
  }
  if (lastStrokeParams.bleed > 0.01) {
    // Use the other accumulation buffer as temp target
    applyBleedEffect(gl, currentTex, width, height, lastStrokeParams.bleed);
    currentTex = writeAccumTex!;
  }
  if (lastStrokeParams.grain > 0.01) {
    applyGrainEffect(gl, currentTex, width, height, lastStrokeParams.grain);
    currentTex = _postTex!;
  }

  // 5. Final blit: render currentTex to screen (gl canvas)
  blitToScreen(gl, currentTex, width, height);

  return _glCanvas;
}
```

**renderStrokeStamps(gl, stroke, w, h):**
1. Bind _strokeFBO, clear to transparent
2. Use _stampProgram (or scatter variant for charcoal)
3. Generate outline points from stroke.points using perfect-freehand's getStroke()
4. Walk along the outline centroid path, placing stamp quads at regular intervals
5. For each stamp: set u_center, u_size (based on stroke.size), u_color (from hexToGLColor), u_hardness (style-dependent: ink=0.7, charcoal=0.3, pencil=0.8, marker=0.9, watercolor=0.2)
6. Use alpha blending for stamp accumulation: `gl.enable(gl.BLEND); gl.blendFunc(gl.SRC_ALPHA, gl.ONE_MINUS_SRC_ALPHA)`
7. Draw each stamp as a triangle strip using _stampVAO

**computeStampPositions(stroke):** Helper that:
1. Calls getStroke(stroke.points, stroke options) to get outline
2. Computes centroid path from the outline (average of corresponding left/right edge points)
3. Returns positions along the centroid at intervals of `stroke.size * 0.3` (denser stamps = smoother)

**compositeSpectral(gl, existingTex, newStrokeTex, targetFBO, w, h):**
1. Bind targetFBO
2. Use _compositeProgram
3. Bind existingTex to texture unit 0 (u_existing)
4. Bind newStrokeTex to texture unit 1 (u_newStroke)
5. Draw fullscreen quad

**applyPostEffect(gl, program, inputTex, targetFBO, w, h, uniforms):**
Generic helper that binds targetFBO, uses program, binds inputTex, sets uniforms, draws fullscreen quad.

**applyGrainEffect / applyBleedEffect:** Wrappers around applyPostEffect with the correct programs and uniforms.

**aggregatePostEffectParams(strokes):**
Takes the max of each FX param across all strokes (since post-effects apply to the entire frame, use the strongest value).

**blitToScreen(gl, tex, w, h):**
Bind framebuffer null (screen), use a simple passthrough program (reuse BRUSH_FX_VERT_SRC + a trivial passthrough frag), bind tex, draw fullscreen quad. OR use gl.blitFramebuffer if available.

Use the passthrough approach: add a PASSTHROUGH_FRAG_SRC in the shader file (or inline):
```glsl
#version 300 es
precision highp float;
in vec2 v_texCoord;
uniform sampler2D u_input;
out vec4 out_fragColor;
void main() { out_fragColor = texture(u_input, v_texCoord); }
```
