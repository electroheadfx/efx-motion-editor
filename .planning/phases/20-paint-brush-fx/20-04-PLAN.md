Create Application/src/lib/brushPreviewData.ts with inline SVG data URIs for brush style preview thumbnails (per D-04: static pre-rendered images, zero runtime cost).

Export `BRUSH_PREVIEW_URLS: Record<BrushStyle, string>` containing inline SVG data URIs for each style. Use SVG for crisp rendering at any DPI.

Each SVG should be ~120x40 viewBox, using `#CCCCCC` as the stroke color on transparent background.

Style-specific SVG characteristics:
- **flat**: Smooth solid `<path>` stroke with round linecap, medium thickness, slight S-curve
- **watercolor**: `<path>` stroke with `<filter>` containing `<feTurbulence type="turbulence" baseFrequency="0.05" numOctaves="3"/>` and `<feDisplacementMap scale="5"/>` for wavy organic edges, lower opacity
- **ink**: `<path>` stroke with varying stroke-width using `<path>` segments, slight `<feTurbulence>` filter for edge variation
- **charcoal**: `<path>` stroke with `<filter>` containing `<feTurbulence baseFrequency="0.6" numOctaves="4"/>` for heavy grain texture
- **pencil**: Thin `<path>` stroke with low opacity (0.5) and fine noise filter `<feTurbulence baseFrequency="0.8"/>`
- **marker**: Wide `<path>` stroke with flat semi-transparent fill (opacity 0.6), no filter, round linecap

Format:
```typescript
import type { BrushStyle } from '../types/paint';

export const BRUSH_PREVIEW_URLS: Record<BrushStyle, string> = {
  flat: `data:image/svg+xml,${encodeURIComponent('<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 120 40">...</svg>')}`,
  watercolor: `data:image/svg+xml,${encodeURIComponent('<svg ...>...</svg>')}`,
  ink: `data:image/svg+xml,${encodeURIComponent('<svg ...>...</svg>')}`,
  charcoal: `data:image/svg+xml,${encodeURIComponent('<svg ...>...</svg>')}`,
  pencil: `data:image/svg+xml,${encodeURIComponent('<svg ...>...</svg>')}`,
  marker: `data:image/svg+xml,${encodeURIComponent('<svg ...>...</svg>')}`,
};
```

Do NOT create actual PNG files. The SVGs should be self-contained with inline filter definitions.
