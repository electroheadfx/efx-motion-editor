**Add private helper method `startAudioPlayback()`:**
```typescript
/** Start all unmuted audio tracks at the correct offset for the current frame */
private startAudioPlayback(): void {
  const currentFrame = timelineStore.currentFrame.peek();
  const fps = projectStore.fps.peek();

  for (const track of audioStore.tracks.peek()) {
    if (track.muted) continue;

    // Compute time offset into the audio based on timeline position
    // Timeline position of audio = track.offsetFrame + track.inFrame
    // Audio content position = (currentFrame - track.offsetFrame - track.inFrame + track.slipOffset)
    const audioFrame = currentFrame - track.offsetFrame - track.inFrame + track.slipOffset;
    const trimDuration = track.outFrame - track.inFrame; // total trimmed frames

    // Only play if current frame is within this track's visible range on timeline
    if (audioFrame >= 0 && audioFrame < trimDuration) {
      const timeOffset = audioFrame / fps;
      // Convert inFrame to seconds for AudioBufferSourceNode offset
      const sourceOffset = (track.inFrame + audioFrame) / fps;
      audioEngine.play(track.id, sourceOffset, track, fps);
    }
  }
}
```

Note: The exact offset calculation depends on how offsetFrame, inFrame, and slipOffset interact. The key formula is:
- The track appears on the timeline starting at frame `(track.offsetFrame + track.inFrame)` for visual purposes (range bar start)
- But for audio playback, we need to compute how far into the AudioBuffer we are
- `timeIntoAudio = (currentFrame - track.offsetFrame) / fps` gives the position relative to the track's local timeline
- `sourceOffset = (track.inFrame + track.slipOffset) / fps + timeIntoAudio` is the actual AudioBuffer position
- The track plays when `currentFrame >= track.offsetFrame` and `currentFrame < track.offsetFrame + (track.outFrame - track.inFrame)`

Simplify: the audio track is audible between timeline frames `track.offsetFrame` and `track.offsetFrame + (track.outFrame - track.inFrame)`. The buffer offset at any given frame is `(currentFrame - track.offsetFrame + track.inFrame + track.slipOffset) / fps`.
