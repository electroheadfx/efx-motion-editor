**Create `Application/src/lib/audioWaveform.ts`:**

Export function `computeWaveformPeaks(buffer: AudioBuffer): WaveformPeaks` that:
1. Mixes down all channels to mono per D-03: for each sample index, average across all channels (`mono[i] = sum(channelData[ch][i]) / numChannels`)
2. Calls internal `extractPeaks(mono: Float32Array, totalLength: number, peakCount: number): Float32Array` for each tier:
   - Tier 1: peakCount = 100 (extreme zoom-out peak envelope)
   - Tier 2: peakCount = 2000 (standard waveform)
   - Tier 3: peakCount = 8000 (zoomed-in detail)
3. `extractPeaks` computes min/max per bucket: `bucketSize = Math.ceil(totalLength / peakCount)`. Output is interleaved `Float32Array(peakCount * 2)` with `[min0, max0, min1, max1, ...]`
4. Returns `{ tier1, tier2, tier3 }`

Import `WaveformPeaks` from `'../types/audio'`.

**Create `Application/src/lib/audioEngine.ts`:**

Export singleton `audioEngine` object (class instance or plain object) with:

**Internal state:**
- `private ctx: AudioContext | null = null`
- `private sources: Map<string, AudioBufferSourceNode> = new Map()` -- keyed by trackId
- `private gains: Map<string, GainNode> = new Map()` -- keyed by trackId
- `private buffers: Map<string, AudioBuffer> = new Map()` -- keyed by trackId

**Methods:**
1. `ensureContext(): AudioContext` -- Lazy-creates AudioContext on first call (per pitfall 1). If ctx.state === 'suspended', calls ctx.resume(). Returns ctx.

2. `async decode(trackId: string, arrayBuffer: ArrayBuffer): Promise<AudioBuffer>` -- calls `ensureContext()`, then `ctx.decodeAudioData(arrayBuffer)`. Caches result in `buffers` map. Returns the AudioBuffer.

3. `getBuffer(trackId: string): AudioBuffer | undefined` -- returns cached buffer.

4. `play(trackId: string, offsetSeconds: number, track: AudioTrack, fps: number): void` -- Creates fresh AudioBufferSourceNode (one-shot per pitfall 2). Connects source -> GainNode -> ctx.destination. Calls `source.start(0, offsetSeconds)`. Applies fade schedule via `applyFadeSchedule()`. Sets volume via gain.gain.value. Stores source and gain in maps.

5. `stop(trackId: string): void` -- Gets source from map, calls source.stop() in try/catch (may already be stopped), disconnects, deletes from maps.

6. `stopAll(): void` -- Iterates all source entries, calls stop() for each.

7. `setVolume(trackId: string, volume: number): void` -- Gets gain from map, sets gain.gain.value = volume.

8. `removeTrack(trackId: string): void` -- Stops playback, removes buffer, source, gain from all maps.

9. `private applyFadeSchedule(gain: GainNode, track: AudioTrack, audioStartTime: number, sourceOffset: number, fps: number): void` --
   Implements gain automation per research Pattern "Gain Fade Schedule Application":
   - Compute fadeInSec = track.fadeInFrames / fps, fadeOutSec = track.fadeOutFrames / fps
   - Compute totalDuration = (track.outFrame - track.inFrame) / fps
   - Compute effectiveEnd = audioStartTime + totalDuration - sourceOffset
   - Cancel any existing scheduled values: gain.gain.cancelScheduledValues(audioStartTime)
   - Apply volume: const vol = track.muted ? 0 : track.volume
   - Fade-in: if fadeInSec > 0 and sourceOffset < fadeInSec, set initial value based on progress, then ramp to vol using exponentialRampToValueAtTime (for 'exponential' curve) or linearRampToValueAtTime (for 'linear'). For 'logarithmic', use linearRamp with a custom curve approximation (setValueCurveAtTime with log-shaped array, or fallback to linearRamp for simplicity).
   - Fade-out: schedule ramp from vol to 0.001 (not 0, because exponentialRamp cannot target 0) at effectiveEnd - fadeOutSec to effectiveEnd.
   - If no fades, just set gain.gain.setValueAtTime(vol, audioStartTime).

Import types from `'../types/audio'` (AudioTrack, FadeCurve).
