## Decisions Made

- Generated interpolation output uses the existing rendered PNG `_frames` storage plus source metadata, avoiding a parallel PNG cache.
- Roto metadata persistence is limited to source/interpolation metadata; no map of per-frame serialized editable projects was added.
- `replaceGeneratedRotoCache()` removes prior generated frames before storing new generated frames so global interpolation setting changes have a clean replacement path.
