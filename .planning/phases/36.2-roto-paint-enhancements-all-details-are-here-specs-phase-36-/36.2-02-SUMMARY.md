# Phase 36.2 Plan 02: Roto Cell Color and Copy Summary

**Standalone Roto workflow cells now distinguish empty gray, cached-only green, and editable-session pink states with save-pending copy that avoids dirty/current-frame color ambiguity.**

## Accomplishments

- Added `RotoCellFill = 'empty' | 'cached-only' | 'editable-session'` and `getRotoCellFill(...)` with tests proving editable session state wins over cached PNG state, cached-only remains green, empty remains gray, and current/dirty state is not a helper input.
- Added `getRotoPendingLabel(...)` so unsaved/saving Roto state is communicated with subtle status text instead of introducing a fourth dirty fill color.
- Updated the workflow strip to accept typed `cachedRotoFrames`, `editableRotoFrames`, `pendingRotoFrames`, `rotoSaveInFlight`, and `onSavePendingRotoFrames` props.
- Filtered `generated-interpolation` entries out of Roto square cell fill calculation so only real-key cache frames affect gray/green/pink square states.
- Replaced old Roto save copy with `Save pending` / `Save current`, disabled when there is no pending dirty Roto frame, while preserving ready/in-flight guards.
- Added semantic Roto CSS fills and kept `.physics-paint-roto-cell.current` to border/outline/box-shadow styling without a background declaration.

## Decisions Made

- Roto dirty state is represented by pending frame membership, dashed styling, and `Unsaved Roto frame pending` / `Saving Roto frame...` status text; it does not alter the semantic cell fill.
- The current frame is outline-only over the existing gray/green/pink fill so it cannot obscure cache/editability meaning.
- Generated interpolation cache frames are intentionally excluded from square-cell semantic fill rendering and left for later connector/cache UI scope.
- The checkpoint was conditionally accepted for this plan's scoped work after user testing: gray/green/pink/save copy may proceed, while missing main EFX Motion cache preview/marker propagation after leaving or closing standalone is recorded as follow-up scope for Plans 36.2-03 and 36.2-04.
