## Deviations from Plan

### Auto-fixed UAT Issues

**1. Copy/Paste state did not become visibly usable after Copy**
- **Found during:** Blocking human UAT.
- **Issue:** Copy produced no clear visual feedback and Paste stayed disabled.
- **Fix:** Kept effectless Copy inside the live session path and added source-contracted pressed states for all key utility buttons.
- **Commits:** `94a13af8`, `ce02ab60`, `305b0d98`.

**2. Source key actions were disabled on a visible real key**
- **Found during:** Blocking human UAT.
- **Issue:** Session availability could be stale while the timeline clearly showed the current frame as a real Roto key.
- **Fix:** WorkflowStrip now falls back to the visible real-key source state for source actions while keeping Paste gated by copied-key state.
- **Commits:** `251ba61d`, `e5d0f700`.

**3. Copy behaved like Cut and Paste produced an empty frame**
- **Found during:** Blocking human UAT.
- **Issue:** A Copy-time canvas snapshot/export path could overwrite the copied payload with the current empty canvas.
- **Fix:** Removed the Copy-time canvas sync helper, made Copy read-only, and preserved copied editable state for Paste transaction application.
- **Commits:** `40421aa6`, `8180938b`.
