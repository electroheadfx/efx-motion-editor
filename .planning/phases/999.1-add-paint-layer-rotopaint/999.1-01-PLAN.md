The undo integration for `addElement`:
```typescript
addElement(layerId: string, frame: number, element: PaintElement) {
  const frameData = this._getOrCreateFrame(layerId, frame);
  frameData.elements.push(element);
  this.markDirty(layerId, frame);
  pushAction({
    undo: () => {
      const f = this._getOrCreateFrame(layerId, frame);
      f.elements = f.elements.filter(e => e.id !== element.id);
    },
    redo: () => {
      const f = this._getOrCreateFrame(layerId, frame);
      f.elements.push(element);
      this.markDirty(layerId, frame);
    },
  });
}
```

Create `Application/src/lib/paintRenderer.ts` that converts PaintElement arrays to Canvas 2D draws using perfect-freehand (per D-01, D-13).

```typescript
import { getStroke } from 'perfect-freehand';
import type { PaintFrame, PaintElement, PaintStroke, PaintShape, PaintFill } from '../types/paint';
```

Export function `renderPaintFrame(ctx: CanvasRenderingContext2D, frame: PaintFrame, width: number, height: number): void`:

For each element in frame.elements, in order:
1. **PaintStroke (tool === 'brush')**: Call `getStroke(element.points, { size: element.size, ...element.options, last: true })` to get outline polygon points. Build a `Path2D`, moveTo first point, lineTo remaining, closePath. Set `ctx.globalAlpha = element.opacity`, `ctx.fillStyle = element.color`, `ctx.fill(path)`.

2. **PaintStroke (tool === 'eraser')**: Same as brush but set `ctx.globalCompositeOperation = 'destination-out'` and `ctx.fillStyle = '#000000'` before filling. Restore composite operation after.

3. **PaintShape (tool === 'line')**: `ctx.beginPath()`, `ctx.moveTo(x1, y1)`, `ctx.lineTo(x2, y2)`, set `ctx.strokeStyle = element.color`, `ctx.lineWidth = element.strokeWidth`, `ctx.globalAlpha = element.opacity`, `ctx.stroke()`.

4. **PaintShape (tool === 'rect')**: If `element.filled`, use `ctx.fillRect(x, y, w, h)` with color. Otherwise `ctx.strokeRect(x, y, w, h)` with strokeWidth. Compute x = min(x1,x2), y = min(y1,y2), w = abs(x2-x1), h = abs(y2-y1).

5. **PaintShape (tool === 'ellipse')**: Use `ctx.ellipse()` centered at midpoint of (x1,y1)-(x2,y2), with radii abs(x2-x1)/2 and abs(y2-y1)/2. Fill or stroke based on `element.filled`.

6. **PaintFill**: Skip rendering here -- fill elements are pre-rasterized as ImageData and stored as a special element type. (Fill rendering will be implemented in Plan 06.)

Each element rendering is wrapped in `ctx.save()` / `ctx.restore()` to isolate alpha and composite operation changes.

Helper function `strokeToPath(points: [number, number, number][], size: number, options: PaintStrokeOptions): Path2D | null`:
- Calls getStroke with the options + { last: true }
- Returns null if outline has < 2 points
- Builds Path2D with moveTo/lineTo/closePath

Export this helper too for reuse in onion skinning preview.
