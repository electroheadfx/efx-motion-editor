**paintFloodFill.ts:**

```typescript
/**
 * Iterative stack-based flood fill on ImageData.
 * Operates in-place on the provided ImageData.
 */
export function floodFill(
  imageData: ImageData,
  startX: number,
  startY: number,
  fillColor: [number, number, number, number],
  tolerance: number = 10,
): void {
  const { data, width, height } = imageData;

  // Clamp start coordinates
  const sx = Math.round(Math.max(0, Math.min(startX, width - 1)));
  const sy = Math.round(Math.max(0, Math.min(startY, height - 1)));

  const startIdx = (sy * width + sx) * 4;
  const targetR = data[startIdx];
  const targetG = data[startIdx + 1];
  const targetB = data[startIdx + 2];
  const targetA = data[startIdx + 3];

  // Don't fill if target matches fill color
  if (
    Math.abs(fillColor[0] - targetR) <= tolerance &&
    Math.abs(fillColor[1] - targetG) <= tolerance &&
    Math.abs(fillColor[2] - targetB) <= tolerance &&
    Math.abs(fillColor[3] - targetA) <= tolerance
  ) {
    return;
  }

  const visited = new Uint8Array(width * height);
  const stack: number[] = [sx, sy];

  while (stack.length > 0) {
    const y = stack.pop()!;
    const x = stack.pop()!;

    if (x < 0 || x >= width || y < 0 || y >= height) continue;

    const pos = y * width + x;
    if (visited[pos]) continue;

    const i = pos * 4;
    if (
      Math.abs(data[i] - targetR) > tolerance ||
      Math.abs(data[i + 1] - targetG) > tolerance ||
      Math.abs(data[i + 2] - targetB) > tolerance ||
      Math.abs(data[i + 3] - targetA) > tolerance
    ) {
      continue;
    }

    visited[pos] = 1;
    data[i] = fillColor[0];
    data[i + 1] = fillColor[1];
    data[i + 2] = fillColor[2];
    data[i + 3] = fillColor[3];

    stack.push(x + 1, y);
    stack.push(x - 1, y);
    stack.push(x, y + 1);
    stack.push(x, y - 1);
  }
}

/**
 * Parse hex color string to RGBA array.
 */
export function hexToRgba(hex: string, opacity: number): [number, number, number, number] {
  const r = parseInt(hex.slice(1, 3), 16);
  const g = parseInt(hex.slice(3, 5), 16);
  const b = parseInt(hex.slice(5, 7), 16);
  const a = Math.round(opacity * 255);
  return [r, g, b, a];
}
```

**OnionSkinOverlay.tsx (per D-08):**

Create `Application/src/components/canvas/OnionSkinOverlay.tsx`. This component renders ghosted paint from adjacent frames ONLY during paint mode when onion skinning is enabled.

The overlay renders onto a canvas element positioned absolutely over the main canvas, with `pointerEvents: 'none'` (clicks pass through to PaintOverlay).

```typescript
import { useEffect, useRef } from 'preact/hooks';
import { paintStore } from '../../stores/paintStore';
import { layerStore } from '../../stores/layerStore';
import { timelineStore } from '../../stores/timelineStore';
import { projectStore } from '../../stores/projectStore';
import { renderPaintFrame } from '../../lib/paintRenderer';
```

The component:
1. Gets a canvas ref
2. On every frame change (timelineStore.currentFrame), or when onion skin settings change, re-renders:
   - Clear the overlay canvas
   - Get current frame, selected paint layer id
   - Loop through previous frames (currentFrame - 1 to currentFrame - prevRange):
     - Get paint data for each frame via `paintStore.getFrame(layerId, frameNum)`
     - If data exists, render with reduced opacity: `baseOpacity * (1 - i / (prevRange + 1))`
   - Loop through next frames (currentFrame + 1 to currentFrame + nextRange):
     - Same opacity falloff formula

Use opacity-only differentiation: render each ghost frame with reduced `ctx.globalAlpha`. The opacity falloff based on frame distance provides sufficient visual distinction between current, near, and far frames. Previous frames use opacity = `baseOpacity * (1 - i / (prevRange + 1))`, next frames use the same formula. No color tinting is applied -- all ghost frames render in their original colors at reduced opacity.

The canvas element must be positioned within the canvas wrapper that already has the CSS transform applied (zoom, pan). It should be a child of the same div that contains the Preview canvas:
```tsx
<canvas
  ref={canvasRef}
  width={projectStore.width.value}
  height={projectStore.height.value}
  style={{
    position: 'absolute',
    top: 0,
    left: 0,
    width: '100%',
    height: '100%',
    pointerEvents: 'none',
  }}
/>
```

Use `useEffect` with dependencies on: `timelineStore.currentFrame.value`, `paintStore.onionSkinEnabled.value`, `paintStore.onionSkinPrevRange.value`, `paintStore.onionSkinNextRange.value`, `paintStore.onionSkinOpacity.value`.
