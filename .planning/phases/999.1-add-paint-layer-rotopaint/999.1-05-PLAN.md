Create `Application/src/lib/paintPersistence.ts` implementing sidecar file persistence (per D-14).

Sidecar file layout:
```
ProjectDir/
  paint/
    {layer-uuid}/
      frame-000.json
      frame-001.json
      ...
```

Each `frame-NNN.json` contains a serialized PaintFrame (the elements array).

Imports:
```typescript
import { readTextFile, writeTextFile, mkdir, exists, readDir, remove } from '@tauri-apps/plugin-fs';
import type { PaintFrame } from '../types/paint';
import { paintStore } from '../stores/paintStore';
```

**Export function `savePaintData(projectDir: string): Promise<void>`:**
1. Get dirty frames from `paintStore.getDirtyFrames()`
2. If no dirty frames, return early
3. For each dirty frame `{ layerId, frame }`:
   a. Ensure directory exists: `await mkdir(projectDir + '/paint/' + layerId, { recursive: true })`
   b. Get frame data: `paintStore.getFrame(layerId, frame)`
   c. If frame data exists and has elements:
      - Pad frame number to 3 digits: `String(frame).padStart(3, '0')`
      - Write: `await writeTextFile(projectDir + '/paint/' + layerId + '/frame-' + paddedFrame + '.json', JSON.stringify(frameData))`
   d. If frame data is null or empty elements:
      - Delete the file if it exists: `const filePath = ...; if (await exists(filePath)) await remove(filePath);`

**Export function `loadPaintData(projectDir: string, layerIds: string[]): Promise<void>`:**
1. For each layerId in layerIds:
   a. Check if paint directory exists: `await exists(projectDir + '/paint/' + layerId)`
   b. If exists, read directory listing: `await readDir(projectDir + '/paint/' + layerId)`
   c. For each file matching `frame-NNN.json`:
      - Parse frame number from filename: `parseInt(filename.match(/frame-(\d+)\.json/)?.[1] ?? '0')`
      - Read file: `const json = await readTextFile(filePath)`
      - Parse: `const paintFrame: PaintFrame = JSON.parse(json)`
      - Load into store: `paintStore.loadFrame(layerId, frameNum, paintFrame)`

**Export function `cleanupOrphanedPaintFiles(projectDir: string, activeLayerIds: string[]): Promise<void>`:**
1. Check if `projectDir + '/paint'` exists
2. If exists, read directory listing to get all layer subdirectories
3. For each subdirectory that is NOT in `activeLayerIds`, delete it recursively
4. This handles cleanup when paint layers are deleted from the project

**Export function `getPaintLayerIds(projectDir: string): Promise<string[]>`:**
1. Check if `projectDir + '/paint'` exists
2. If exists, return subdirectory names (these are layer IDs)
3. If not, return empty array

Note on error handling: Wrap all file operations in try/catch. Log errors via console.error but don't throw -- paint persistence failure should be non-fatal (similar to how Phase 16 handles audio pre-render failure).
