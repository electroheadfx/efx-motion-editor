## Summary

Replaced the flat 2-column grid in ShortcutsOverlay with a tabbed interface. Added Sections index tab as table of contents with 7 group tabs (Playback, File, Editing, Navigation, Canvas, Blur, Keyframes) and full keyboard navigation.

## What Was Done

- Refactored ShortcutsOverlay.tsx from 2-column grid to tabbed layout
- Sections index tab (tab 0) shows all 7 groups with entry counts, clickable to jump
- Number keys 1-7 jump directly to group tabs from any tab
- Arrow Left/Right cycle through tabs with wrapping
- Tab key advances to next tab (same as Arrow Right)
- Arrow Up/Down navigate rows in Sections index, Enter jumps to that group
- Modal min-height 320px prevents height jumps between tabs
- Overlay resets to Sections tab on reopen
- All colors use CSS variables, no hardcoded hex values

## Outcome

ShortcutsOverlay is now organized by category with full keyboard navigation, ready to accommodate shortcuts from future phases 13-17.
