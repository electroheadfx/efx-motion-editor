Create `Application/src/lib/glBlur.ts` (~150-200 lines) implementing a WebGL2 two-pass separable Gaussian blur with the following structure:

**Shader sources (inline template literals):**

Vertex shader (GLSL ES 3.00, `#version 300 es`):
- `layout(location = 0) in vec2 a_position` attribute
- `out vec2 v_texCoord` varying
- Maps clip space [-1,1] to texture space [0,1]: `v_texCoord = a_position * 0.5 + 0.5`

Fragment shader (GLSL ES 3.00, `#version 300 es`, `precision highp float`):
- `uniform sampler2D u_texture`
- `uniform vec2 u_direction` -- `(1.0/width, 0.0)` for horizontal, `(0.0, 1.0/height)` for vertical
- `uniform float u_radius` -- blur radius in pixels
- `uniform bool u_preserveAlpha`
- Gaussian weight function: `exp(-0.5 * i*i / (sigma*sigma))` where `sigma = max(u_radius / 3.0, 1.0)`
- Dynamic loop up to `int(ceil(u_radius))`, capped at 64 iterations for GPU safety
- Samples center + symmetric pairs in both directions, accumulates weighted color, normalizes by total weight
- When `u_preserveAlpha` is true: output `vec4(color.rgb, centerAlpha)` using the unblurred center-pixel alpha
- When false: output full blurred `color`

**Module-level cached state:**
- `_gl: WebGL2RenderingContext | null = null`
- `_glCanvas: HTMLCanvasElement | null = null`
- `_resources: BlurResources | null = null`
- `_initFailed: boolean = false`

**BlurResources interface:**
```typescript
interface BlurResources {
  program: WebGLProgram;
  texSource: WebGLTexture;
  texIntermediate: WebGLTexture;
  fbo: WebGLFramebuffer;
  vao: WebGLVertexArrayObject;
  uTexture: WebGLUniformLocation;
  uDirection: WebGLUniformLocation;
  uResolution: WebGLUniformLocation;
  uRadius: WebGLUniformLocation;
  uPreserveAlpha: WebGLUniformLocation;
  width: number;
  height: number;
}
```

**`getGL()` function:**
- Returns cached `_gl` if available
- Returns null if `_initFailed` is true
- Creates canvas via `document.createElement('canvas')`
- Calls `getContext('webgl2', { premultipliedAlpha: false, preserveDrawingBuffer: true, antialias: false, depth: false, stencil: false })`
- If null, sets `_initFailed = true`, returns null
- Registers `webglcontextlost` event listener on the canvas that sets `_gl = null` and `_resources = null` (lazy re-init on next call)
- Caches and returns the GL context

**`compileShader(gl, type, source)` helper:**
- Creates shader, sets source, compiles
- Checks `gl.getShaderParameter(shader, gl.COMPILE_STATUS)`
- On failure: `console.warn('glBlur shader compile:', gl.getShaderInfoLog(shader))`, deletes shader, returns null

**`createResources(gl, width, height)` function:**
- Compiles vertex + fragment shaders via `compileShader()`
- Creates and links program, checks `LINK_STATUS`
- Gets all uniform locations: `u_texture`, `u_direction`, `u_radius`, `u_preserveAlpha`
- Creates VAO with fullscreen quad vertices: `[-1,-1, 1,-1, -1,1, 1,1]` as `TRIANGLE_STRIP`
- Creates two textures (`texSource` and `texIntermediate`) using `texStorage2D(TEXTURE_2D, 1, RGBA8, width, height)` with `CLAMP_TO_EDGE` wrap and `LINEAR` min/mag filter
- Creates one framebuffer, attaches `texIntermediate` as `COLOR_ATTACHMENT0`
- Checks framebuffer completeness via `checkFramebufferStatus`
- Returns `BlurResources` object with tracked width/height
- On ANY GL error during setup: `console.warn`, cleanup, return null

**`ensureResources(gl, width, height)` function:**
- If `_resources` exists and dimensions match, return it
- If dimensions changed: delete old textures, framebuffer, program (via `gl.deleteTexture`, `gl.deleteFramebuffer`, `gl.deleteProgram`), set `_resources = null`
- Call `createResources(gl, width, height)`
- Cache in `_resources` and return

**`applyGPUBlur()` -- the public export:**
```typescript
export function applyGPUBlur(
  source: HTMLCanvasElement,
  targetCtx: CanvasRenderingContext2D,
  radiusNorm: number,
  width: number,
  height: number,
  preserveAlpha: boolean,
): boolean
```
- Returns `false` if GPU path unavailable (caller uses CPU fallback)
- Calls `getGL()`, returns false if null
- Resizes `_glCanvas` to match width/height if needed
- Calls `ensureResources(gl, width, height)`, returns false if null
- Converts normalized radius to pixel radius using `normalizedToPixelRadius` (import from fxBlur or inline the same formula: `Math.max(0, Math.round(radiusNorm * radiusNorm * Math.max(width, height) * 0.05))`)
- If pixelRadius < 1, returns false (no blur needed)
- Upload source: `gl.bindTexture(TEXTURE_2D, texSource)`, `gl.texSubImage2D(TEXTURE_2D, 0, 0, 0, RGBA, UNSIGNED_BYTE, source)`
- Pass 1 (horizontal): bind FBO, viewport(0,0,w,h), useProgram, set `u_direction = (1/w, 0)`, `u_radius = pixelRadius`, `u_preserveAlpha`, bind texSource, bindVAO, `drawArrays(TRIANGLE_STRIP, 0, 4)`
- Pass 2 (vertical): bind FBO=null (render to screen), viewport(0,0,w,h), set `u_direction = (0, 1/h)`, bind texIntermediate, `drawArrays(TRIANGLE_STRIP, 0, 4)`
- Readback: `targetCtx.drawImage(_glCanvas!, 0, 0, width, height)`
- Return `true`

```typescript
export function applyBlur(
  canvas: HTMLCanvasElement,
  ctx: CanvasRenderingContext2D,
  radius: number,
  width: number,
  height: number,
  preserveAlpha: boolean,
): void {
  if (radius <= 0) return;

  // Try GPU path first
  if (applyGPUBlur(canvas, ctx, radius, width, height, preserveAlpha)) {
    return;
  }

  // CPU fallback: StackBlur
  const pixelRadius = normalizedToPixelRadius(radius, Math.max(width, height));
  if (pixelRadius < 1) return;
  if (preserveAlpha) {
    canvasRGB(canvas, 0, 0, width, height, pixelRadius);
  } else {
    canvasRGBA(canvas, 0, 0, width, height, pixelRadius);
  }
}
```
