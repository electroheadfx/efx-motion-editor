**CollapseHandle.tsx changes:**

1. Replace `import { GripVertical } from 'lucide-preact'` with `import { Tally3 } from 'lucide-preact'`.

2. Replace the inner icon rendering. Currently there is a single `<GripVertical>` centered in the div. Replace with 3 stacked `Tally3` icons arranged vertically in a column. The icon wrapper div should:
   - Use `flex flex-col items-center justify-center gap-0` layout (vertical stack, tight spacing with gap-0)
   - Each `Tally3` icon: `size={14}`, rotated 90 degrees via `style={{ transform: 'rotate(90deg)' }}` so the tally lines appear horizontal (like horizontal grab lines)
   - Apply the transition and hover opacity classes to the wrapper div, not individual icons

Keep `group-hover:opacity-100` on the wrapper with a base `opacity-70` class (instead of inline 0.5) so hover still brightens it.

The JSX for the icon area should look like:
```tsx

     style={{ color: 'var(--sidebar-collapse-line)' }}>
```
