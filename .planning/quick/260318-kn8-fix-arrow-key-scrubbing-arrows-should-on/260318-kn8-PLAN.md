Fix arrow key scrubbing so arrows are context-dependent based on mouse cursor position:
- Mouse over timeline: arrows scrub timeline (Left/Right step frame, Up/Down also step)

3. Create a new helper function `handleArrow`:
```typescript
function handleArrow(axis: 'x' | 'y', delta: number): void {
  const region = uiStore.mouseRegion.peek();

  if (region === 'timeline') {
    // Timeline region: arrows always scrub timeline
    // For x-axis: use delta directly (negative = backward, positive = forward)
    // For y-axis: treat up as backward, down as forward
    if (axis === 'x') {
      if (delta < 0) playbackEngine.stepBackward();
      else playbackEngine.stepForward();
    } else {
      // Up (delta < 0) = step backward, Down (delta > 0) = step forward
      if (delta < 0) playbackEngine.stepBackward();
      else playbackEngine.stepForward();
    }
    return;
  }

  if (region === 'canvas') {
    // Canvas region: nudge selected layer if any
    nudgeIfSelected(axis, delta);
    return;
  }

  // Fallback ('other'): left/right scrub timeline, up/down nudge layer
  if (axis === 'x') {
    const selectedId = layerStore.selectedLayerId.peek();
    if (selectedId) {
      nudgeIfSelected(axis, delta);
    } else {
      if (delta < 0) playbackEngine.stepBackward();
      else playbackEngine.stepForward();
    }
  } else {
    nudgeIfSelected(axis, delta);
  }
}
```

4. Update the four ArrowLeft/ArrowRight handlers and Shift variants to call `handleArrow('x', ...)` instead of `nudgeOrStep('x', ...)`.

5. Update the four ArrowUp/ArrowDown handlers and Shift variants to call `handleArrow('y', ...)` instead of `nudgeIfSelected('y', ...)`.

Wait, cleaner approach: the delta already encodes magnitude. For timeline context, the step size IS the delta magnitude:

```typescript
if (region === 'timeline') {
  // delta encodes both direction and magnitude (e.g., -10 for Shift+Left)
  const step = axis === 'x' ? delta : (delta < 0 ? -Math.abs(delta) : Math.abs(delta));
  const current = timelineStore.currentFrame.peek();
  playbackEngine.seekToFrame(Math.max(0, current + step));
  return;
}
```

Actually even simpler -- delta is already signed with the right magnitude:
```typescript
if (region === 'timeline') {
  const current = timelineStore.currentFrame.peek();
  // For y-axis: up (negative delta) = backward, down (positive delta) = forward
  // delta already has correct sign and magnitude (1 or 10 with shift)
  playbackEngine.seekToFrame(Math.max(0, current + delta));
  return;
}
```

This works because:
- ArrowLeft calls handleArrow('x', -1) -> seeks to current - 1
- Shift+ArrowLeft calls handleArrow('x', -10) -> seeks to current - 10
- ArrowUp calls handleArrow('y', -1) -> seeks to current - 1
- ArrowDown calls handleArrow('y', 1) -> seeks to current + 1
- Shift+ArrowUp calls handleArrow('y', -10) -> seeks to current - 10

Use this simplest form.
